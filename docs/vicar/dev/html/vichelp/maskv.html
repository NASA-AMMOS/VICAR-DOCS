Help for MASKV

 MASKV formats images for hardcopy display. This includes the 
 numbering of lines and samples, a grey wedge along the right side,
 and, optionally, Vicar labels, logos, and a job id.

 RESTRICTIONS:

 MASKV only displays up to 70 characters per label line.
 Any label that exceeds 70 characters will be truncated to 70 characters
 on the hard copy.

PARAMETERS:

INP

 input data set

OUT

 output data set

SIZE

 image size

SL

 starting line

SS

 starting sample

NL

 number of lines

NS

 number of samples

COMP

 complement the input image VALID: COMP

STRETCH

 apply a linear stretch to the input image

EXPAND

 enlarge the characters in the output by this factor

ID

 ID's displayed VALID: ID, NOID

LOGO

 LOGO's displayed VALID: LOGO, NOLOGO

LLOGO

 left logo number (1-5)

RLOGO

 right logo number (1-5)

TASK

 display task names from the VICAR labels VALID: TASK, NOTASK

USER

 display user name and date- time from the VICAR label VALID: USER, NOUSER

HISTORY

 display VICAR history labels VALID: HISTORY, NOHISTOR, TIMS

SYSTEM

 display VICAR system labels VALID: SYSTEM, NOSYSTEM
