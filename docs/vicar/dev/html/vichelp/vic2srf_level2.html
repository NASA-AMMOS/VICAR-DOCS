Level 2 Help for VIC2SRF

INP

 
INP specifies the name or names of the input file(s).  Color images may
be either three separate files, entered as red, green, blue, or a single
multichannel file.  If a multichannel file is input, the BANDS parameter
must also be used.  Currently, the input is restricted to BYTE data.
BIP organization is not supported.

SROUT

SROUT specifies the name of the output file, which will be in SunRaster
format.  For black-and-white images, the image will be stored as 8-bit.
For color images 24 bit storage will be used.

BANDS

The BANDS parameter is used when the input dataset is a multichannel
image.  It is used to designate which band or bands are to be displayed.
For color images, order the bands - red, green, blue.
