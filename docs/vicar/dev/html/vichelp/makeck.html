Help for MAKECK

PURPOSE:  MAKECK creates a new SPICE C-kernel as follows:

	MAKECK  OUT=filename

where "filename" is the filename of the output C-kernel.

PROGRAM HISTORY:

Original programmer: Gary Yagi
Cognizant programmer: Gary Yagi
Revisions:  New

PARAMETERS:

OUT

 String Output C-kernel
