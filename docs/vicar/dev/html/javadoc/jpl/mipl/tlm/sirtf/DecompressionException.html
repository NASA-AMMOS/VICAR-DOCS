    public class DecompressionException
extends InvalidPacketException
    An exception that is thrown when a failure occurs while trying to decompress a SIRTF science packet. SIRTF uses a Rice-base compression algorithm for the science packets. Invariably, any errors in decompression can be assumed to be due to FULDL errors.
