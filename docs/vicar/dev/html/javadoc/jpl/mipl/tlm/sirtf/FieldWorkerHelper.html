Class FieldWorkerHelper

    • jpl.mipl.tlm.sirtf.FieldWorkerHelper

    WorkerHelper

    public class FieldWorkerHelper
extends java.lang.Object
implements WorkerHelper
    Replaces the necessary elements of the TPS system with fieldable elements. This class is mainly used for those cases where a CORBA TPS system isn't available of feasible (e.g. ATLO at Lockheed). This class provides basic, non-TPS resources that substitute for those typically found within the TPS. Some of these things are:
    • an SFDU source (from a file only at the moment)
    • a UPF-handling emulator since the TPS provides the UPF as a Properties object already

    SirtfTlmProc will use this class instead of the TPS supplied version when running in the standalone configuration.
