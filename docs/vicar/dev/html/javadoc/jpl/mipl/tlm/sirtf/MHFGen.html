Class MHFGen

  • java.lang.Object
    • jpl.mipl.tlm.sirtf.MHFGen

  • public class MHFGen
extends java.lang.Object
    MHFGen is a standalone program that creates the momentum history file Supplement files as described in SIS FES-014. This application is intended to be run from the command line by a cron.

    The required parameters are:

    • UPF - the UPF. Used to obtain database connection info
    • start SCLK - the beginning SCLK for the MH file
    • version - the version of the MH file set

    MHFGen retrieves the salient telemetry data from the SIRTF Pointing database via a stored procedure. MHFGen then time-aligns the data and creates any fill records to account for gaps in time.
