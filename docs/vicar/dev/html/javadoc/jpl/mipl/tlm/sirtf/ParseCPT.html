jpl.mipl.tlm.sirtf

Class ParseCPT

  • java.lang.Object
    • jpl.mipl.tlm.sirtf.ParseCPT

  • public class ParseCPT
extends java.lang.Object
    A parser that ingests the Channel Parameter Table, populating an internal TreeMap with the desired channel information. Also, a main() method is provided which vauses the parser to zip through the CPT and prints to standard out the following format:
    channel ID|TITLE|description|units
    This format is used by the C++ sirtftlmproc so we didn't have to incorporate all this parsing code there.
