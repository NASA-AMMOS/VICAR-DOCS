Package jpl.mipl.tlm.sfdu Description

Java classes for processing Multimission SFDU Packets. This package contains classes that are useful for manipulating and parsing SFDU's. Generic CHDO and SFDU classes are supplied, as well as, SIRTF specific CHDO's. Other projects should be able to quickly use or subclass these in order to work with their data.

A generic SFDU reader can be obtained by using the SFDUReader class' main() function. Simply pass a file of SFDU as the single command line argument and logging information will describe the contents. This has been successfully run for both SIRTF and Cassini.
