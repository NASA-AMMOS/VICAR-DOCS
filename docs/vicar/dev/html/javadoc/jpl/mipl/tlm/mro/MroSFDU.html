    SFDU

    public class MroSFDU
extends GenericSFDU
    A class that encapsulates a MRO SFDU for SHARAD (apid = 7) A MRO SFDU consists of following:
    • the primary CHDO is of type 2
    • the secondary CHDO is of type 90, and
    • the tertiary CHDO is of type 86.

    However, this class encapsulates three basic SFDU types: data, CDR, or ECDR. For each of these types, the following must also be true for a valid SFDU:
    • for a MRO data SFDU (Standard Packets for channelization)
      • the quaternary CHDO does not exist
      • the data CHDO is of type 10
    • for a MRO CDR (Channelized Data Record) or
      • the quaternary CHDO is of type 27
      • the data CHDO is of type 28
    • for a MRO ECDR (Expanded Channelized Data Record)
      • the quaternary CHDO is of type 32
      • the data CHDO is of type 29

    None of the current MRO processing uses CDR's. This class tries to obtain the same information with the getXXX() methods whether or not the source is a data or ECDR SFDU.
