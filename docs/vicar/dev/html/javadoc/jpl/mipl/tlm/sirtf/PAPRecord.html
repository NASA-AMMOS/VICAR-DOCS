jpl.mipl.tlm.sirtf

Class PAPRecord

  • java.lang.Object
    • jpl.mipl.tlm.sirtf.PAPRecord

  • public class PAPRecord
extends java.lang.Object
    Bean to encapsulate salient information to tell PAP which packets not to delete from the spacecraft MMC (Mass Memory Card). Such packets will then be retransmitted, giving us a second shot at them. Presumably, the initial packets were corrupted due to a known memory corruption issue -- colloquially called a "FULDL hit" (pronounced foo-dul).
