    public abstract class SirtfFileProduct
extends java.lang.Object
implements SirtfOutputProduct
    Provides a great deal of common behavior for the standard products MIPL is responsible for. DCE products have a naming convention, specified in SIS FOS-SIS-3010, that includes various items, some of which are not part of this objects original key name. A good example of this is the version number. Also, for IRS, the channel number is not part of the key. Therefore, the final name is created and used after it has been determined that this class should close out ond post-process the file.
