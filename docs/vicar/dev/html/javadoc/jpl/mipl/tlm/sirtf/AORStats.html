Class AORStats

  • java.lang.Object
    • jpl.mipl.tlm.sirtf.AORStats

  • public class AORStats
extends java.lang.Object
    Convenience class to allow for the tracking of Lists of products that are related by their AORkey. Four Lists are maintained:
    1. complete, expedited products
    2. partial, expedited products
    3. complete, nominal products
    4. partial, nominal products
