Class FieldWorkerHelper

  • java.lang.Object
    • jpl.mipl.tlm.mro.FieldWorkerHelper

  • public class FieldWorkerHelper
extends java.lang.Object
    Replaces the necessary elements of the TPS system with fieldable elements. This class provides basic, non-TPS resources that substitute for those typically found within the TPS. Some of these things are:
    • an SFDU source (from a file only at the moment)
    • a UPF-handling emulator since the TPS provides the UPF as a Properties object already
