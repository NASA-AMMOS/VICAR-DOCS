Interface EDRReader

  • All Known Implementing Classes:
    EDRLoader, FitsEDRReader, IsisEDRReader, VicarIoEDRReader

    public interface EDRReader
    Defines the method(s) required to comply with an EDR reader class. The constructor for any <file>Reader will/must have have the following signature:
    <file>Reader ( String file )
    where file is the input EDR filename.

    Unfortunately, this doesn't help in the case where the input might be from a stream. I'll have to think about this and make the constructor more general.
