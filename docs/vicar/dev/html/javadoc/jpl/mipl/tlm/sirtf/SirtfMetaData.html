Class SirtfMetaData

  • java.lang.Object
    • jpl.mipl.tlm.sirtf.SirtfMetaData
  • All Implemented Interfaces:
    java.io.Serializable

    public class SirtfMetaData
extends java.lang.Object
implements java.io.Serializable
    Encapsulates a set of characteristics describing a SirtfFileProduct. Such statistics are usually recorded into some kind of persistent storage like a database or a serialized object. They might also be used in reports describing the processing results.

      • getMissingPacketGroupings

        public java.util.List getMissingPacketGroupings()
        Return a List which contains pairs of entries for each packet gap. The first value of the pair is the starting packet (0-based) for the gap. Consecutive packets in a gap are total for the second value of the pair. Thi smethod is used by the static constructMissingLineStats() method for IRAC, IRS, and MIPS.

        public void resetForReprocess()
        Reset many properties preserving version and ECDR tracking information. When reprocessing, we want the version to remain unique. Also, we currently can't "best" ECDR entries in the FITS header, so ECDR tracking is kept from previous runs as well.
