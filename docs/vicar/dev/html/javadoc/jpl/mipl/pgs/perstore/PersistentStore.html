    public class PersistentStore
extends ActivatableServerBase
implements IntrPerStore_IF
    Purpose: This class is an implementation of the IntrPerStore_IF interface. This class defines new methods that corresponds to the methods defined in PersistentStore_IF that are called only after certain conditions are meant (e.g., user is authenticated). It also allows listeners to register with this class for listening to changes to a particular table. This class will load the project specific perstore implementation and call its methods to perform the perstore functionality. See the project specific implemenations for the details of what each method does.
