Interface UserInformation_IF

  • All Superinterfaces:
    java.rmi.Remote
    All Known Subinterfaces:
    IntrPerStore_IF, PersistentStore_IF
    All Known Implementing Classes:
    Database, DivPerstoreImpl, MarsPerstoreImpl, McsPerstoreImpl, PersistentStore

    public interface UserInformation_IF
extends java.rmi.Remote
    Purpose: This interface is used for retrieving information about the users: both personal information and configuration information. The user's personal information includes items such as the full name, email and pager number. The user's configuration information captures the relationship between the user and his given permissions. It is possible that one user is authorized to run requests for multiple instruments, and for each instrument, the user may have different permissions.
