                  java.lang.String gapName,
                  boolean keepHeader)
           throws java.lang.Exception
        Filter out debug and dump packets.
        Parameters:
        inputPktFileName - name of input packet file
        SciFileName - name of output science packet file
        InstDumpFileName - name of output debug packet file
        DREBFileName - name of output dump packet file
        gapName - name of output gap file
        keepHeader - if set to false, the headers are not included in report file.
        Returns:
        an int value of a four bit binary number, left shifted four bits, representing which files have been generated, as follows. Bit 1 (left most bit) represent science file, bit 2 instrument dump, 3 DREB dump and bit 4 represents gap file. So a int value 160 (binary 10100000) means both science and DREB dump file are generated. Any other value means an error.
        Throws:
        java.lang.Exception - if there an error reading or writing SFDU files.
