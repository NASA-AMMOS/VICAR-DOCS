Package com.fg.xample Description

This is MATIS adaptation of XAmple. XAmple is a GUI XML Editor. MATISXAmple changes the default window actions and also loads the default MATIS schema and configuration files. XAmple loads some resources which are assumed to be in com.fg.xample package. Also, some of the fiels in XAmple class are package protected. As a result, MIPLXAmle has to be in com.fg.xample package.
