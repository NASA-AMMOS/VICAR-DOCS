    public class TaskRunner
extends ProcessController
implements org.jbpm.graph.def.ActionHandler
    Executes the given task. The taskClassName argument of this class can be used to specify a class of type Task that performs some work. The specified task class is executed synchronously. Refere to AsyncTaskRunner for asynchronous task execution.

      • taskCompleted

        public void taskCompleted(boolean normalTermination)
                   throws java.lang.Exception
        Used by task to indicate that it has completed.
        Parameters:
        normalTermination - True means that task completed normally and thus the token runner should continue signaling. A value of false means that this task is not complete and in event of rerun, the execution should start at this node. Remember the task calling this method is responsible for process contolling (i.e., stopping it, pausing it, etc.) in event of abnormal termination.
        Throws:
        java.lang.Exception - if an error is encountered while performing any of the above steps
