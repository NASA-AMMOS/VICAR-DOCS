Class UPF

  • java.lang.Object
    • jpl.mipl.pgs.utils.UPF

  • public class UPF
extends java.lang.Object
    Allows parsing of User Preference File. 
     
      Copyright 2002, California Institute of Technology.
      ALL RIGHTS RESERVED.
      U.S. Government Sponsorship acknowledge.2002.
 
      MIPL Telemetry Processing Subsystem
  
 
  The following default values are used when parsing the UPF:
    # - comment. Ignore reset of the line after this symbol
    keyword=value - for single valued keyword. Eg. MISSION=SIRTF
    keyword=(value1,value2,...,valueN) - for N valued keyword.
    Eg. DN_TO_EU_COEFF=(2.34, 6.12,0.98).
    + - only allowed in the end, but before an end comment.
        Marks line continuation. Eg. MISSION=+
    INSTRUMENT or MISSION=+ # this is test
    INSTRUMENT.
    Version:
    1.0
    Author:
    Raj Patel {rrp@jpl.nasa.gov}, Cecilia Cheng {csc@mipl.jpl.nasa.gov}
