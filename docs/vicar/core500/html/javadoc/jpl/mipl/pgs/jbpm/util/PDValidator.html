Class PDValidator

  • java.lang.Object
    • jpl.mipl.pgs.jbpm.util.PDValidator

  • public class PDValidator
extends java.lang.Object
    This class is used by ProcessRunner to validate given process definition before executing it. This class checks that each path reaches end state (by traversing each path), that each path in m/split meets correct m/join, that there are no m/split or m/join in a loop, etc.
