Class ServerDependencyDetector

  • java.lang.Object
    • jpl.mipl.pgs.gui.launcher.ServerDependencyDetector

  • public class ServerDependencyDetector
extends java.lang.Object
    This class detects a dependency on servers. Certain server needs to be started before some other server starts, and it also depends on a status of one server to another. This class detects such dependency.
