Class TokenService

  • java.lang.Object
    • jpl.mipl.pgs.jbpm.services.TokenService

  • public class TokenService
extends java.lang.Object
    A utility class used by jpl.mipl.pgs.jbpm package to perform token related tasks. This class is used to send/receive token related messages that are internal to the PGS system.
