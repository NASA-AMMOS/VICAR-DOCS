jpl.mipl.pgs.utils.FEIUtils

Interface FEIErrorHandler


  • public interface FEIErrorHandler
    This interface is to be implemented by classes that are interested in handling exception that may occure while adding file(s) to FEI. NOTE: The exception that is passed to this handler is not logged by FEI. Thus, the implementing class should log it, if so desired.
