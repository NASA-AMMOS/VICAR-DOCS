    Class RTCanvas: This class is the root of all RealTime Canvas Widgets. These include: EDRCanvas, HIstogramCanvas, KeywordCanvas ImageCanvas, and others. This class establishes the PopupMenu action as well as other base functions that all *Canvas objects need.
