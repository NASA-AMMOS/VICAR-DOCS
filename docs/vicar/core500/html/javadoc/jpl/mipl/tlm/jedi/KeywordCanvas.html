    public class KeywordCanvas
extends RTCanvas
    This class allows for the display of the particular EDR label contents that are of interest. Instantiation is usually accomplished by supplying an EDR to be observed. Since KeywordCanvas inherits from RTCanvas (and is therefore a PropertyChangeListener), then all EDR changes should propogate to change the displayed label item.

    This class also allows for an alarm to be set such that a color change will be made to the displayed text under configurable circumstances. The following comparisons, if met, will change the color of the displayed value accordingly:

    • NONE This is the default. Nothing will cause an alarm condition until this setting is changed.
    • value is EQUAL_TO alarm condition
    • value is NOT_EQUAL_TO alarm condition
    • value is GREATER_THAN alarm condition
    • value is LESS_THAN alarm condition
    Of course, the last two conditions only apply if the label item and the alarm value are interpreted as numbers and not strings.
