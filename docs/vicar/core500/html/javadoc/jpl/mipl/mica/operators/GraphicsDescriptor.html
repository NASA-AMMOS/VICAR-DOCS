Class GraphicsDescriptor

  • All Implemented Interfaces:

    public class GraphicsDescriptor
extends javax.media.jai.OperationDescriptorImpl
implements java.awt.image.renderable.RenderedImageFactory
    The Graphics JAI operator applies a GraphicsDrawList in a JAI Rendered chain. The GraphicsDrawList is passed in via a ParameterBlock.

    This operator forms a bridge between the JAI and Java2D packages. It functions similarly to the JAI RenderedGraphics operator, except that it can be inserted at any point in a JAI chain and not just as a source. The overhead incurred by this operation includes the overhead of the GraphicsDrawList plus a Raster to Raster copy.

    This operator is limited in the same manner as other JAI to Java2D bridges which allow JAI objects to be inserted into a Java2D rendering. Both this Java2D into JAI operator and JAI into Java2D approaches involve wrapping the JAI image in a BufferedImage. The Java2D BufferedImage is limited to TYPE_BYTE, TYPE_INT, and TYPE_USHORT image datatypes.
