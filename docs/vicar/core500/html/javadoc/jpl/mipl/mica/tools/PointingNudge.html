    public class PointingNudge
extends javax.swing.JPanel
implements PointingPlugin
    A PointingNudge is a PointingPlugin that provides functionality to nudge a pair of orthogonal pointing parameters with a keypad. These values are also displayed in a list. Any nudge or edits cause the associated ImageChip to be reprojected. This plugin is used if an orthogonal pair of parameters can be found such as azimuth and elevation or right ascension and declination.
