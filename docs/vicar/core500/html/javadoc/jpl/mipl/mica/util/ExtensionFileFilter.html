Class ExtensionFileFilter

    java.io.FileFilter

    public class ExtensionFileFilter
extends javax.swing.filechooser.FileFilter
implements java.io.FileFilter
    This utility class is used by a JFileChooser to filter the list of files it displays. FileFilters can be added to the JFileChhoser and it will provide these filter options to the user so that they can show only the subset of a directory that includes the files they wish to see.

    Instances of this utility can be added to provide filtering based on a file's extension. It can also be used to create filter to show only VICAR image files. 

     
     // Show all files that have ".lis" somewhere in their name
        ExtensionFileFilter listfilter=new ExtensionFileFilter();
        listfilter.add(".lis");
        listfilter.setContains(true);
        listfilter.setDescription("List files (*.lis*)");
     
     // Show all VICAR files
        ExtensionFileFilter vicarfilter=new ExtensionFileFilter();
        vicarfilter.addVicar();
        vicarfilter.setDescription("Vicar image files");
     
     // Show all files that end with ".gif"
        ExtensionFileFilter imagefilter=new ExtensionFileFilter();
        imagefilter.add(".gif");
        imagefilter.setDescription("GIF Image files");
     
     JFileChooser fileChooser = new JFileChooser();
        fileChooser.addChoosableFileFilter(listfilter);
        fileChooser.addChoosableFileFilter(vicarfilter);
        fileChooser.addChoosableFileFilter(imagefilter);

     int returnValue = fileChooser.showOpenDialog (null);
     if (returnValue == 0)  {  // User selected OK
        File selectedFile = fileChooser.getSelectedFile();
     }
