    public class Editor
extends javax.swing.JFrame
implements java.awt.event.ActionListener
    The Editor is used to start the MICA system, initialize its major components, then handle the global graphical interface. The Editor main method bootstraps all the various packages that MICA depends on, including JPL and MICA native libraries, Java Advanced Imaging, and the JPL supplied codec for the Vicar image file format. Visually, the Editor holds the MICA menus, and a split-pane for the global Display and global ToolPalette. Behind the scenes, it also instantiates the ImageBag and loads any imagery specified on the command line.
