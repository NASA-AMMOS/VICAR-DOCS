    public class Selection
extends javax.swing.JPanel
implements java.beans.PropertyChangeListener
    A Selection tool is a JavaBean that manages a selection of ImageChips. The tool allows ImageChips to be added and removed from an ImageBag selection in various ways. It also provides ways to annotate the selected ImageChips with border highlighting.

    The selection interface is a simple property editor for an ImageBag selection. It also annotates ImageChips by inserting RenderedOps onto their ImageChains. Further, it uses an InvisibleOverlay to receive mouse click information from a global Display object.
