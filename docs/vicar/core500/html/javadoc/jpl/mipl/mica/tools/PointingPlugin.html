Interface PointingPlugin

  • All Known Implementing Classes:
    PointingList, PointingNudge, PointingTwist

    public interface PointingPlugin
    A PointingPlugin is an interface specifying the functionality that the Plugin tool requires of components it uses to display sets of pointing parameters.
    Author:
    SGT, Inc.
    See Also:

      • setSelection

        void setSelection(java.util.Vector v)
        Set the selection of ImageChips that are to be displayed. A plugin can decide if it will display all chips or only the first in the selection.

      • setPointingTool

        void setPointingTool(Pointing pt)
        Give the plugin a reference to the Pointing tool that it will reside in. Used to inform the tool that the user wishes to use a different pointing parameter for one of the two mouse "drag & point" axes.

      • setParameterIndicies

        void setParameterIndicies(int first,
                          int last)
        For a given ImageChip, which of its pointing parameters should this plugin display. All parameters and their names must be displayed between the first and last index, inclusive. The parameters may be obtained from the ImageChip's PointingModel.
