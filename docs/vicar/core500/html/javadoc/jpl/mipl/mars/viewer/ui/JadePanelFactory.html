Class JadePanelFactory

  • java.lang.Object
    • jpl.mipl.mars.viewer.ui.JadePanelFactory

  • public class JadePanelFactory
extends java.lang.Object
    Purpose: Factory for IJadePanels. RdrImageNode's may contain a variety of images (most commonly one or two images), so this factory inspects the node and based on the current configuration determines what implementation of the IJadePanel interface to instantiate. Currently supported are mono and stereo image sets.
