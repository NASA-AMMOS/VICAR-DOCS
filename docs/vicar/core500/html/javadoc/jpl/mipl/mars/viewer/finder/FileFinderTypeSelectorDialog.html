    public class FileFinderTypeSelectorDialog
extends javax.swing.JDialog
    Purpose: Visual component providing user with way of specifying file finder type and root for creation of a new MarsImageFileFinder instance. The constructors expect an instance of FileFinderRegistry, which contains metadata on various file finders available to application. Ensure that a registry is properly initialized before providing it to this class.
