Class FileResourceMetadataCache

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.FileResourceMetadataCache

  • public class FileResourceMetadataCache
extends java.lang.Object
    Purpose: More of a work-around to maintain file metadata from remote sources, this is a static lookup cache that maintains a most recent set of N entries containing a file location and its associated metadata (modification time, file size).
