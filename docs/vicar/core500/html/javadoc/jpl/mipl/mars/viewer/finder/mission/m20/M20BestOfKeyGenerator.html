    public class M20BestOfKeyGenerator

    For M20, the claim is you can collect a set of filenames with a common key, sort alphanumerically, and then select the last entry to represent the best of the set. The base implementation creates a common key by removing the following fields: camera-specific, downsample, compression, producer, version.
