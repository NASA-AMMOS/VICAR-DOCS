    public class PhxTieredGroupIdentifier
extends java.lang.Object
implements GroupIdentifier
    Purpose: Proxy for Phoenix group identifier. Attempts to retrieve information from database source, if available. If unsuccessful, then informaion is retrieved from the image label (with a greater performance hit). Debugging Note: You can enable debug messages that print the result and source of group id retrivals by including the following system property: phx.group.id.debug This should *ONLY* be used to quickly assess database accessibility.
