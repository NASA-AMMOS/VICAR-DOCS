Class SurrogateUnderlySelector

    MerUnderlySelector, MslUnderlySelector, NsytUnderlySelector

    public abstract class SurrogateUnderlySelector
extends DefaultUnderlySelector
    Purpose: Extends the default underly selector with extra behavior to select a surrogate source product if a source product with expected overlay id does not exist. This is an abstract class, and extenders must override getSurrogateOverlayId() to provide a default overlay id based on the original to use for lookup as a surrogate.
