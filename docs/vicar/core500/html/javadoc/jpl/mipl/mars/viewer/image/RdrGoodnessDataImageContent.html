    public class RdrGoodnessDataImageContent
extends RdrValueHueMapImageContent
    Purpose: Encapsulates the JAI render graph and controls for a goodness product that has data values in the other bands, inspired by M20 rougheness product. TODO: data values will need to be coverted to RGB Properties: @see RdrFormatImageContent, @see RdrValueHueMapImageContent
