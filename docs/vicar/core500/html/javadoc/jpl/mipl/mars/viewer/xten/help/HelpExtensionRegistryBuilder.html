jpl.mipl.mars.viewer.xten.help

Class HelpExtensionRegistryBuilder

  • java.lang.Object
    • jpl.mipl.mars.viewer.xten.help.HelpExtensionRegistryBuilder

  • public class HelpExtensionRegistryBuilder
extends java.lang.Object
    Purpose: Builder for HelpExtensionRegistry instances. By creating a builder using the static method builder(), client can pass the resulting builder object an XML-file defined according to the Mars Image Help Extension definition to load specific help extension information for use with the Mars Image Viewer.
