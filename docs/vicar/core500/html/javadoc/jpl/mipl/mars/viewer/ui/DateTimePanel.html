    public class DateTimePanel
extends javax.swing.JPanel
    Purpose: The DateTime Panel is a GUI component where the user can specify a date/time value using either (1) YYYY-MM-ddThh:mm:ss format, or using the number of milliseconds since the epoch of January 1, 1970, 00:00:00 GMT. Updating one field will update all others.
