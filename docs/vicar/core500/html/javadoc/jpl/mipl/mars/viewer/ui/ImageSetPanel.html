    public class ImageSetPanel
extends javax.swing.JPanel
implements ImageContentObserver, ProductSetPanel
    Purpose: Component that displays an image set as a tabbed pane of image displays. Note: Each panel has an active state that must be activated and deactivated based on whether or not the panel is being viewed. This controls propagation of panel state to the application (i.e. the app model should only be notified that the panel's activeNode has been set if the panel is active)

      • getNewSelectedIndex

        protected int getNewSelectedIndex(RdrImageSet oldSet,
                                  RdrImageSet newSet,
                                  int lastSelectedIndex)
        Returns the index from the newSet parameter that should be displayed first. If the previous set had an active image of type found in the new set, then the first matching index will be returned. Otherwise, the DEFAULT_INDEX will be returned. If newset is empty, then NULL_INDEX is returned. If either set is null, then NULL_INDEX is returned.
        Parameters:
        oldSet - RdrImageSet containing the last selected image
        newSet - RdrImageSet that may contain the same type as last selected.
        lastSelectedIndex - Index that was selected last from old set
        Returns:
        Index to be selected from new set
