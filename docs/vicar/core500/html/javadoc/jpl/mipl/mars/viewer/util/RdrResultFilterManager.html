Class RdrResultFilterManager

  • All Implemented Interfaces:
    PropertyManager

    public class RdrResultFilterManager
extends PropertyManagerImpl
    Purpose: This manager was introduced to handle filtering of the application's RdrResult based upon various filter criteria. Components can either listen to the main application model for unfiltered results or this model for the filtered results. This model implicitly listens to the main model for updates to filter criteria and source RdrResult.
