Class DisparityGridExtractor

    Purpose: Utility class that takes a 2-banded map and produces a set of points and optionally lines that form a grid of the data. This grid can be based on data availability or the data values themselves.

        protected boolean isInvalidData(double[] data,
                                double invalidValue)
        If all values in data array are equal to the invalidValue, then if is assumed there is no data for that sample. In which case, true would be returned. If at least one entry in the data is not equal to the invalid value, then it is assumed that this is real data.

        True if samples are invalid, false otherwise

      • getDataAt

        protected boolean getDataAt(int x,
                            int y,
                            double[] output,
                            GridExtractionParameters parameters)
        Get the data for the x,y location in the image and store in the output array. If (x,y) are legitimate coordinates and there is valid data at that location, then the output value will be updated with the data and true will be returned. If (x,y) is invalid, or there is invalid data at that location, false will be returned.
        Parameters:
        x - x-coordinate of pixel
        y - y-coordinate of pixel
        output - Output data array, should be of size two or more

      • calculatePoint

        protected java.awt.geom.Point2D calculatePoint(int x,
                                               int y,
                                               double[] dispData,
                                               GridExtractionParameters parameters)
        Calculates the point that will be part of the grid for location (x,y) based on the whether "mapped data drawing" or "relative disparity" are enabled. If mapped data, then we use the data values at (x,y), added to (x,y) if relative disparity. Otherwise we use (x,y) itself. NOTE: Disparity maps use the first band for line (y-value), and second band for sample (x-value).
        Parameters:
        x - x-coordinate
        y - y-coordinate
        dispData - Disparity data values
