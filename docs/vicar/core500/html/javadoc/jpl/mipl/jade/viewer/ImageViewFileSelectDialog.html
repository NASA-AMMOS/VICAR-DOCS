    public class ImageViewFileSelectDialog
extends javax.swing.JDialog
    Purpose: Panel used to select an image, image type, and optional associated source image file. Can return image data as a string via the filename (if a single file), or a RenderedOp that results from combining the mulitple images into a single image (ie, for images corresponding to single bands). This class was modified to accept a configuration file enumerating image names and associated types instead of the class having direct knowledge. If no config is used, then this dialog allows images to be viewed generically (no special behavior). A default configuration file is located within the config subpackage. This can be overridden by specifying a value for the jade.image.config property when invoking JVM, i.e. java ... -Djade.image.config=myConfig.xml ...
