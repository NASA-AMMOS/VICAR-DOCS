Class MouseScroller

    public class MouseScroller
extends javax.swing.event.MouseInputAdapter
    A component that implements mouse scrolling for a JViewport. It responds to click-hold-drag of the mouse (left button) over the viewport and scrolls it appropriately. It can be attached to any JViewport but is usually used with JScrollPane on an image display. However, because it generates diagonal scroll events (simultaneous X and Y changes), you may want to use it with JViewportImage.

    TBD: add property to control whether the scroll is constrained to the display or not.
