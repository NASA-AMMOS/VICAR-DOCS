Help for ASTERTIR

PURPOSE:

   ASTERTIR is a program that converts a TIMS ITEM, GTEM, or BTEM image to
a simulation of an ASTER TIR image, in radiance units.

PARAMETERS:

INP

 TIMS temperature image file, (BTEM, ITEM, or GTEM)

OUT

 ASTER TIR radiance image file, 5 channels, BIL format.

SIZE

 The standard VICAR output size field.

SL

 Starting line

SS

 Starting sample

NL

 Number of lines

NS

 Number of samples
