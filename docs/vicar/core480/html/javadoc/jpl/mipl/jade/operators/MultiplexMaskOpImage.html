    public class MultiplexMaskOpImage
extends javax.media.jai.PointOpImage
    Purpose: A JAI operator which is used to extract multiplexed data via a bit-shift and bit-mask operation upon a source image. Operation: Bit-Shift Parameters: 1) shiftValue (bit-shift) and 2) maskValue (bit-mask) Output: pixelOut = (pixelIn >> shiftValue) & maskValue Operation: Maximum-Or Parameters: 1) widthValue (bit-width) 2) sampleFlags (sample-flags) Output: pixelOut = Max( set of active pixels of size width packed in the integer) This active set is determined by the sampleFlags if set, or includes all pixels if the sampleFlags is not set. A MultiplexMaskOpImage should not be instantiated directly, but through the JAI operation registry.

                            int operationId,
                            boolean[] maxFlags)
        Constructs a MultiplexMaskOpImage. The image dimensions are copied from the source image. The tile grid layout, SampleModel, and ColorModel may optionally be specified by an ImageLayout object.
        Parameters:
        source - a RenderedImage source
        layout - an ImageLayout optionally containing the tile grid layout, SampleModel, and ColorModel, or null.
        shift - Bit shift value. Pixel value will be shifted a number of bits to the right.
        mask - Bit mask value. A logical-and op will be applied to the result of the bit-shift op.
        width - Bit width value. For maximum or operation, indicates the width of each value within the integer DN.
        packetCount - Number of bits in the multiplexed packet
        operationId - Indicates the operation to be performed, either shift-mask or max-or operation.
        maxFlags - Array of booleans indicating which samples to use in the maximum-or operation.
