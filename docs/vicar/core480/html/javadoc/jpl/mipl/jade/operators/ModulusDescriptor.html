Class ModulusDescriptor

    public class ModulusDescriptor
extends javax.media.jai.OperationDescriptorImpl
implements java.awt.image.renderable.RenderedImageFactory
    Purpose: The Modulus operator takes one rendered source image, an array of range mins, and range maxs, applies a modulus function type of clamp on pixel values. If the range is exceeded, then the resulting value will be a circular sequence, so that max+1 results in min. The destination pixel values are defined by the pseudocode: 
      diff = range_max - range_min;
  curVal = src[x][y][b]
 
  while (curVal > range_max)
     curVal = curVal - diff;
  while (curVal < range_min)
     curVal = curVal + diff;
  dst[x][y][b] = curVal
