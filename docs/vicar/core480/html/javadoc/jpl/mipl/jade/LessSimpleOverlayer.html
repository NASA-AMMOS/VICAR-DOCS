Class LessSimpleOverlayer

  • java.lang.Object
    • jpl.mipl.jade.LessSimpleOverlayer

  • public class LessSimpleOverlayer
extends java.lang.Object
    Purpose: This program accepts a primary image, its type, and a background image. The primary image is then overlaid upon the background image based on its type and written to file.
