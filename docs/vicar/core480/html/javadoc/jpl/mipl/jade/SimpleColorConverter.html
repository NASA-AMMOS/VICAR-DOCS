jpl.mipl.jade

Class SimpleColorConverter

  • java.lang.Object
    • jpl.mipl.jade.SimpleColorConverter

  • public class SimpleColorConverter
extends java.lang.Object
    Purpose: This program accepts three images (or a three-band image), decomposes it to HSI space, manipulates HSI values, converts back to RGB, and writes result. Current image manipulation is adding/subtracting constants from HSI bands.
