    public class AwsServiceInfoBase
extends java.lang.Object
implements AwsServiceInfo
    Purpose: Base service for interacting with AWS S3 data repositories representing OPGS data stores. We require the following information: 1) AWS region; 2) AWS S3 bucket name; 3) Relative path from bucket root. We support the following service root locations: 1) Path-Style AWS S3 URLs Example URL: http://s3-us-gov-west-1.amazonaws.com/m20-ids-marsviewer-testdata/M20_data/ocs/phase/sol/ Expr: {protocol}://s3-{region}.amazonaws.com/{bucket}/{path} 2) Virtual-hosted AWS S3 URLs Example URL: http://m20-ids-marsviewer-testdata.s3-us-gov-west-1.amazonaws.com/M20_data/ocs/phase/sol/ Expr: {protocol}://{bucket}.s3-{region}.amazonaws.com/{path}
