jpl.mipl.mars.viewer.finder.util

Class HistoryModelSync

  • java.lang.Object
    • jpl.mipl.mars.viewer.finder.util.HistoryModelSync

  • public class HistoryModelSync
extends java.lang.Object
    Purpose: Convenience class that creates a FileFinderHistory model and syncs it with the application model. Specifically, the text representation of the history is persisted in the model. When the model uses a new file finder, its type and root are added to the history. This, in turn, will update the history and result in a new persisted entry for the model.
