    public class RdrTerrainClassV2ImageContent
extends RdrValueHueMapImageContent
    Purpose: Encapsulates the behavior for terrain class types, a 2 or N banded product where first band indicates the value of something... The hues are (out of 360): [120, 240, 60, 30, 0, 180] which should correspond to colors [green, blue, yellow, orange, red, cyan]. Properties: @see RdrTerrainClassImageContent
