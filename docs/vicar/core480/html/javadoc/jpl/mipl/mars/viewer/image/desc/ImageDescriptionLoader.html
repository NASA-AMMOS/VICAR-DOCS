Class ImageDescriptionLoader

  • java.lang.Object
    • jpl.mipl.mars.viewer.image.desc.ImageDescriptionLoader

  • public class ImageDescriptionLoader
extends java.lang.Object
    Purpose: Loader creates new ImageDescriptions instance based on a set of configuration lookup rules, checking for a valid configuration in the following order: (1) User-provided File (constructor) (2) File associated with property miv.desc.config (3) A default configuration file.
