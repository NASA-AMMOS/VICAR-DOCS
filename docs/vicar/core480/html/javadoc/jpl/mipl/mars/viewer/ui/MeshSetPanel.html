        protected boolean updateRdrs(RdrMeshSet newSet)
        Receives a List of RDR paths or RdrImageContents and creates a collection of RDR images. Sets internal collection reference to this new collection. Called from outside the class after a collection of rdrs has been retrieved. RDR tabbed pane will then be refreshed to display RDRs
        Parameters:
        newRdrSet - Vector of filepath strings for RDR images, or vector of preconstructed RdrImageContent objects
        Returns:
        True if successful update occurred, false otherwise.

      • getNewSelectedIndex

        protected int getNewSelectedIndex(RdrMeshSet oldSet,
                                  RdrMeshSet newSet,
                                  int lastSelectedIndex)
        Returns the index from the newSet parameter that should be displayed first. If the previous set had an active image of type found in the new set, then the first matching index will be returned. Otherwise, the DEFAULT_INDEX will be returned. If newset is empty, then NULL_INDEX is returned. If either set is null, then NULL_INDEX is returned.
        Parameters:
        oldSet - RdrImageSet containing the last selected image
        newSet - RdrImageSet that may contain the same type as last selected.
        lastSelectedIndex - Index that was selected last from old set
        Returns:
        Index to be selected from new set

        protected void setActiveNodeIndex(int index,
