    public class MarsImageRdrView
extends javax.swing.JPanel
    Purpose: Panel used to display the tabbed pane of RDR's. Contains a collection of JadeDisplays, one per image to be displayed. Included are GUI controls for Zoom, Overlay, Operator Controls, Line/Sample display, and Dn display.
