jpl.mipl.mars.viewer.ui

Class FrameFactory

  • java.lang.Object
    • jpl.mipl.mars.viewer.ui.FrameFactory

  • public class FrameFactory
extends java.lang.Object
    Purpose: Factory that returns instances of JFrame, configured for a specific display mode (i.e. stereo, standard) Because the marsviewer and jadeviewer applications will have a single frame for the lifetime of the application, global properties that a particular mode require may be set. (i.e. stereo mode turns off double buffering for the repaint manager). These side-effects break away from the strict factory pattern.
