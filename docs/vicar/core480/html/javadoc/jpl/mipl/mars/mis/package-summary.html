Package jpl.mipl.mars.mis

  • Interface Summary 
    Interface Description
    Command
    Interface that declares and documents all of the supported commands.
    Constants
    Constants file for Mars Image Sampler service
  • Class Summary 
    Class Description
    Connection
    Contains all of the methods and properties associated with a single websocket connection.
    FileManager
    Manages a global list of FileRefs and ensures a unique read in of each filepath from memory.
    FileRef
    Contains the read-in file data associated with a given image file as well as methods for extracting pixel data and life-span control.
    MISApplicationConfig
    This class provides setup data to the server hosting environment (Tomcat) for how the Application should be deployed and registeres the endpoint classes so that they may be accessed by clients.
    MISEndpoint
    This class provides a concrete object for instantiation by the server for each connection requested by a client.
