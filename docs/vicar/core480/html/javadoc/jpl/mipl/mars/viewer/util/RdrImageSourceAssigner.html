Class RdrImageSourceAssigner

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.RdrImageSourceAssigner

  • public class RdrImageSourceAssigner
extends java.lang.Object
    Purpose: Handling source node assignment was getting too complicated for the RdrImageSet class, so it was moved to this one. This class can now focus on determining the source nodes for all overlay ids, including providing a substitute source for nodes that do not have a valid source product.
