jpl.mipl.mars.viewer.finder.preset

Class FinderPresetRegistryBuilder

  • java.lang.Object
    • jpl.mipl.mars.viewer.finder.preset.FinderPresetRegistryBuilder

  • public class FinderPresetRegistryBuilder
extends java.lang.Object
    Purpose: Builder for FinderPresetRegistry instances. By creating a builder using the static method builder(), client can pass the resulting builder object an XML-file defined according to the Mars Image File Finder Preset definition to load specific file finder information for use with the Mars Image Viewer.
