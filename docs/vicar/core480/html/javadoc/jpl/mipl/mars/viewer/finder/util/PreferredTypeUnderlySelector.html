Class PreferredTypeUnderlySelector

  • java.lang.Object

    Direct Known Subclasses:
    M20MosaicUnderlySelector, M20UnderlySelector, NsytMosaicUnderlySelector

    public abstract class PreferredTypeUnderlySelector
extends DefaultUnderlySelector
    Base class that extends the default behavior of the underly selector by giving preference to certain product types. If none of those are present in a product list, then the default selection behavior is used. Missions will extend this to supply their specific product types.
