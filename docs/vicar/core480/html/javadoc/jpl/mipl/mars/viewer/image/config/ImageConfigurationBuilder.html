jpl.mipl.mars.viewer.image.config

Class ImageConfigurationBuilder

  • java.lang.Object
    • jpl.mipl.mars.viewer.image.config.ImageConfigurationBuilder

  • public class ImageConfigurationBuilder
extends java.lang.Object
    Purpose: Builder for ImageTypeInfo instances. By creating a builder using the static method builder(), client can pass the resulting builder object an XML-file defined according to the Jade Image Info definition to loaded specific image information for use with the Jade Image Viewer.
