    public class HistogramCanvas
extends RTCanvas
    Class HistogramCanvas: This class is designed to allow for the display of the values held in the Histogram (or Statistics) class. In this way, I've divorced the view of the data from the model of the data.

    The HistogramCanvas will display the histgram within an image area defined by 

    x_size
    and 
    y_size
    in the constructor call (default is 256 x 100). It will also display the mean and standard deviation. The mean being shown as a red line, and the standard deviation as hilighted histogram lines (+/- one sigma).

    Additionally, the HistogramCanvas object, by extending my RTCanvas class, allows for the interactive selection of the vertical axis: Linear or Log.
