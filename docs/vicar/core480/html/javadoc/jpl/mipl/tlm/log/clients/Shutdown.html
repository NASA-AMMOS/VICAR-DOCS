Class Shutdown

  • java.lang.Object
    • jpl.mipl.tlm.log.clients.Shutdown

  • public class Shutdown
extends java.lang.Object
    Shut down a log server. This executable class tells a MIPL Log Server to shut down using the shutdown method.

    To run this class, you must specify the system property jpl.mipl.tlm.log.objectName, which tells what MIPL Log Server to contact. You'll also need to specify the IOR of the naming server with the -ORBnaming ior command-line argument or with the ooc.service.NameService=ior property.
