      • JediStreamListener

        public JediStreamListener()
        Sets up a thread to listen for TPS events. Contains a ConsumerStreamEvent class which receives StreamStarted, StreamStopped and ProductAvailable events. StreamStarted event sets up a new StreamObject in the StreamManager, and starts a StreamTimer. If no client is listening to the stream after (5) minutes, the stream is removed. StreamStopped simply sends a message to the client informing it that the stream has stopped. ProductAvailable events add the product reference to the StreamObject if an appropriate StreamObject exists.
