Class PDS4Metadata

  • java.lang.Object
    • javax.imageio.metadata.IIOMetadata
      • jpl.mipl.io.plugins.PDS4Metadata
  • All Implemented Interfaces:
    java.lang.Cloneable

    public class PDS4Metadata
extends javax.imageio.metadata.IIOMetadata
implements java.lang.Cloneable
    Version:
    0.5
    The metdata object is needed by an IIOImage to be a container for the metadata for an image. The reader will create the Document and put it in here. If the "common" IIOMetadataNode version of the metedata is requested it will then be built using the native Document as the input. VicarBinaryLinePrefix has been added to support HRSC images which do contain Binary prefix data. The metadata is used to transfer this data since prefix data is normally not read for an image to be displayed.
