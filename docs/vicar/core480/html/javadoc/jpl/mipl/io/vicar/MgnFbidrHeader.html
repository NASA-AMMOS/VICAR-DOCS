      • readHeader

        public void readHeader(java.io.DataInput di)
                throws java.io.IOException
        Actually read the header. Assumes we're given a DataInput object that reads and translates the proper VAX/VMS real/float formats (e.g. the return from VicarDataFormat.getDataInputWrapper()).

        Exceptions are simply returned, and abort the read.

        No sanity checking is done, e.g. on the contents of the header or types. What you read is what you get.

        Throws:
        java.io.IOException
