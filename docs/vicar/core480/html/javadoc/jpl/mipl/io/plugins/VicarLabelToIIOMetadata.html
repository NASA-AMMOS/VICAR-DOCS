Class VicarLabelToIIOMetadata

  • java.lang.Object
    • jpl.mipl.io.plugins.VicarLabelToIIOMetadata

  • public class VicarLabelToIIOMetadata
extends java.lang.Object
    This class builds a IIOMetadata from a vicar image label object.
    A later version MAY handle reading a String which contains the label contents and parses that text. This will be called from inside a reader.
