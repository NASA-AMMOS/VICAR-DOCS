Class PDSLabelToDOM

  • java.lang.Object
    • jpl.mipl.io.plugins.PDSLabelToDOM

  • public final class PDSLabelToDOM
extends java.lang.Object
    Parses a ISIS image label and creates a Document Object with the values. Using the JDK 1.4 String Matcher and Pattern methods instead.
    Version:
    $Id: PDSLabelToDOM.java currently does NOT handle HISTORY part of the label
    to add HISTORY we need to get the label size, read the whole thing into a buffer and process the buffer.
    Author:
    Steve Levoe
