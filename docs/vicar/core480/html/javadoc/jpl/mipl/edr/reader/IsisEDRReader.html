Class IsisEDRReader

    • jpl.mipl.edr.reader.IsisEDRReader

    EDRReader

    public class IsisEDRReader
extends java.lang.Object
implements EDRReader

    A reader for Cassini ISS VIMS cubes. All bands in the cube are saved in a Map object, keyed by the virtual index band number, along with all properties and tasks.

    Original author is Myche McAuley written for JEDI. This class is then extracted from JEDI.

        public void loadFile()
              throws java.io.IOException
        loadFile will ingest the entire contents of the ISIS label and the image data. Since I'm assuming that RealTime images *tend* to be on the order of 1k x 1k, this shouldn't be a problem. I don't know what I'll do when EDRs grow to be 100 Mbytes each ;-) --MISR for example--. But, one step at a time.

        loadFile does not currently support binary headers or prefixes.
