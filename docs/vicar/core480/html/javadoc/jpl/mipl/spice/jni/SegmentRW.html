Class SegmentRW

  • java.lang.Object
    • jpl.mipl.spice.jni.SegmentRW

  • public class SegmentRW
extends java.lang.Object
    A segment of a SPICE E kernel which may be updated. Note that there are two segment classes. Use Segment when you want to add a whole segment at once. Use SegmentRW when you want to add to, delete from, or modify an existing segment.
