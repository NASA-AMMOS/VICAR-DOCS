    public class PointingTwist
extends javax.swing.JPanel
implements PointingPlugin
    A PointingTwist is a PointingPlugin that provides functionality to nudge a pair of orthogonal pointing parameters with a keypad and allow for twist correction. These values are also displayed in a list. Any nudge or edits cause the associated ImageChip to be reprojected. This plugin is used if an orthogonal pair of parameters can be found such as azimuth and elevation or right ascension and declination. This code was taken from PointingNudge.java by SCT, Inc. and modified for the twist and other enhancements.
