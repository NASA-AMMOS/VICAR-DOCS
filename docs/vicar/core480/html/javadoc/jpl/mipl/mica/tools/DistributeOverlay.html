    public class DistributeOverlay
extends InvisibleOverlay
    A DistributeOverlay assists a Distribute tool interfacing with an output mosaic. This object maintains two anchor ImageChips and (when added to a Display object) draws anchor spots on each along with a "tie-line" between them. The obejct also provides methods for determining if other ImageChips intersect the tie-line and at what point.

      • intersect

        public double intersect(ImageChip test,
                        ImageChip origin)
        Determines if a test ImageChip intersects a line between the two anchor chips. The method returns the distance from the origin anchor to the test chip as a value between 0.0 (test chip exactly overlapping the origin chip) and 1.0 (test chip as far from the origin anchor as the other far anchor is). Returns -1.0 if there was no intersection.
