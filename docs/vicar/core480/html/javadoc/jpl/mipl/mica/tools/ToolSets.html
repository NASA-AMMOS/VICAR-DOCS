    public class ToolSets
extends javax.swing.JPanel
implements javax.swing.event.TreeModelListener
    A ToolSets tool is a JavaBean that manages the loading of Java Archive (JAR) files with sets of tool beans. The tool interfaces with the a global jpl.mipl.mica.ToolPalette object and uses it to load additional tool sets, remove tool sets, and save tool set preferences to the MICA configuration file.

    It registers itself with the ToolPalette to get notified of changes in the tool tree. This can happen via e.g. the Load Toolset menu item.
