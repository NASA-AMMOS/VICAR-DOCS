      • getLoadFactor

        float getLoadFactor(java.util.Map request)
        Get the load factor based on the request that is passed in.
        Parameters:
        request - The request for which the load factory is requested.
        Returns:
        A float value between 0.0 and 1.0 indicating the current load of the factory. 1.0 means the factory is running at it's max capacity. Return CANNOT_HANDLE_THIS_REQUEST to indicate that the ServiceFactory this load calculator is associated with cannot run the given request now.
