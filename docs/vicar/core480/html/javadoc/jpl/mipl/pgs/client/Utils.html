      • parseCmdArgs

        public static java.util.Map<java.lang.String,java.lang.String> parseCmdArgs(java.lang.String[] args)
        Parses command line arguments looking for any key that starts with -. Returns a Map with keys as key. The following example (-u, rrp, -p, password, -e, -f, -whatisthis, thisistest) will return the following map 
          <I>Key          Value</I>
  -u             rrp
  -p           password
  -e             true
  -f             true
  -whatisthis  thisistest
