    public class FileFilterConfiguration
extends ConfigurationBase
    The file filter configuration is used by {#link DirEventListener DirEventListener} to obtain it's various queues. A DirEventListener can have more than one FileFilterConfigurations but one of those can be the primary file filter (i.e. is_trigger can only be true for only one of the file filter)
