Help for PHOTFIT2M_GENERAL

PURPOSE:
This menu point is dedicated to input general parameters for the program such 
as the names of input photometric catalog IBIS file, the type of 
photometric funtion, the fit conditions and the kind of outputs.

NOTE : The parameter PHO_FUNC is in need to select the photometric function :  
When returning to the highest level of the menu you will see that the second 
selection point has been changed according to your input of PHO_FUNC in this 
menu.

PARAMETERS:

INP

 Input IBIS files, containing the points generated by PHOTOM

PHO_FUNC

 Photometric function type

CLASS_ID

 Class-id

NORM

 Causes subroutine Metropolis to renormalize itself.

RERUN

 Number of rerun of metropolis

MAXITER

 Specifies the total number of successful iterations.

NUMTEN

 Specifies the number of successful iterations which must be accumulated before the width of the solution generating probability function drops by a factor of ten.

PERCENT

 The minimum acceptable # of points with residuals below tolerance.

TOLERANC

 The I/F residual tolerance. !.VARIABLE PLOT !Keyword for plotting the residuals. !NOPLOT deactivates plotting.

PRINT

 Keyword for printing of the IBIS input files. NOPRINT deactivates printing. !.VARIABLE DISP !Disp on image display?

METROP

 Causes subroutine Metropolis to list the iteration progress.
