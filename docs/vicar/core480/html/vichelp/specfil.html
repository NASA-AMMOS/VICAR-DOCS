Help for SPECFIL

     This program produces a low pass box filtered image in the spectral
dimension for data in MSS or any of the 3-D formats. If the input is a 3-D
file with BSQ organization, on output it will be changed to BIL organization.

PARAMETERS:

INP

 Input file name

OUT

 Output file name

SIZE

 Window into input

BANDS

 Window into input in band dimension

SL

 Starting line = size(1)

SS

 Starting sample = size(2)

SB

 Starting band = bands(1)

NL

 Number of lines = size(3)

NS

 Number of samples = size(4)

NB

 Number of bands = bands(2)

MSS

 Number of MSS bands, if MSS format

WEIGHTS

 Width of filter
