Help for SPECTRUM

PURPOSE:
SPECTRUM produces a plot of DN versus channel or wavelength for a single 3-D 
pixel, or of the mean DN for a region specified by one or more rectangular
areas. The plot may be directed to an x-window (via IDL), or to a
PostScript printer.

WRITTEN BY:  Ron Alley,  May 1989
COGNIZANT PROGRAMMER:  Ron Alley
REVISION: 3 - November 8, 1994

PARAMETERS:

INP

 Input image file, Optional wavelength file

BANDS

 Starting band, # of bands

SB

 Starting band

NB

 Number of bands

LINE

 Line number of plotted pixel

SAMPLE

 Sample number of plotted pixel

XAXIS

 Range of x-axis (bands/wavlen)

YAXIS

 Range of y-axis (DN's)

AREA

 (SL1,SS1,NL1,NS1,..., SLn,SSn,NLn,NSn)

SENSOR

 MASTERTIR, TM, AVIRIS, TIMS, GEOSCAN, 87AVIRIS, 90AVIRIS

XLABEL

 (for laser printer only) label for for x-axis

YLABEL

 (for laser printer only) label for for y-axis

ST_CHAN

 Channel number of first channel to be plotted

TITLE

 Title to be placed on pen plot

CUMUL

 Plot the cumulative function instead of the spectrum

PRINT

 Print a table of the values? (PRINT or NOPRINT)

SIGMA

 Plot the 1-std deviation envelope also? (SIGMA and NOSIGMA)

EXCLUDE

 Exclude zero valued pixels? (EXCLUDE or NOEXCL)

MODE

 use NOPLOT to suppress plotting and pauses, use WINDOW to plot in Xwindow, use PSCRIPT for a hardcopy plot
