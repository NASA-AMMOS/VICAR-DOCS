Help for TIMSCONV

PURPOSE:

   TIMSCONV is a program which converts TIMS data from radiance to temperature,
or vice versa.  The input must be a 3-D file of all 6 bands, and the
output will also be a 3-D file of all 6 bands, though the user may choose
the output format (byte, half, full, or real). 

EXECUTION:

   The following is the execution statement format for TIMSCONV:

	TIMSCONV INP=PIX OUT=CALPIX PARAMS

   where INP, OUT, and PARAMS are parameters discussed in their res-
pective parameter sections.

PARAMETERS:

INP

 TIMS radiance or temperature image.

OUT

 TIMS temperature or radiance image; opposite of INP

SIZE

 The standard VICAR output size field.

SL

 Starting line

SS

 Starting sample

NL

 Number of lines

NS

 Number of samples

MODE

 Conversion is to these units Valid: TEMP, RAD

TEMPFAC

 Rescaling factor to get image data into degrees Celsius

RADFAC

 Rescaling factor to get image data into milliwatts/(m*m*sr*micrometer)

FORMAT

 Output data format Default is same as input format Valid: BYTE, HALF, FULL, REAL

EMIT

 The emittance assumed for each band.

DATE

 Date of data acquisition. Used to override the date in the VICAR label (yymmdd)
