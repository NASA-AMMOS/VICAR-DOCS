Help for WEDGE


PURPOSE:

"WEDGE" is a VICAR applications program which creates a "pie" wedge
image. The wedge can be a simple pie wedge or a "bowtie".

EXECUTION

    wedge out=wedge.img  size=(1,1,256,256) stangle=5.0 arc=5.0 x0=128 y0=128

This will create a wedge from 5.0 to 10.0 and 185.0 to 190.0 degrees

 RESTRICTIONS:
  1. BYTE DATA ONLY. Follow WEDGE with cform to convert to other formats
  2. STANGLE and ARC must be less than 180 degrees
  3. Bowtie option still not available
  4. Maximum size of image is 2048x2048

PARAMETERS:

OUT

 An output data set

SIZE

 The standard Vicar size field

NL

 Number of lines

NS

 Number of samples

STANGLE

 Starting angle less than 180 degrees

ARC

 less than 180 degrees

DNMIN

 - Minimum dn - Default = 0

DNMAX

 INTEGER-OPTIONAL - Maximum DN (0-255) - Default=255

X0

 KEYWORD-OPTIONAL - Sample coordinate of spot center relative to 1,1 - Default = 0 (NS/2)

Y0

 KEYWORD-OPTIONAL - Line coordinate of spot center relative to 1,1 -Default =0 (NL/2)

KEYWORD-optional Bowtie or Nobowtie reflect wedge around x0,y0 -Default = No bowtie
