Help for MSSIBIS

PURPOSE

    "mssibis" transfers data between MSS format images and IBIS tabular file.
    Data may be transfered to IBIS files from Vicar images using a mask image.



EXECUTION


mssibis  INPUT.MSS  OUTPUT.INT   SIZE=(100,50,160,200) INC=4   		MSS=4 BANDS=(1,3,4)  COLS=(1,3,5) NCOL=7 'TOIBIS

mssibis  INPUT.MSS  OUTPUT.INT   MSS=4

mssibis  INPUT.INT  OUTPUT.MSS 'TOMSS  COLS=(2,4,6,7) NL=50 NS=60

mssibis  (INPUT.MSS,MASK.IMG)  OUTPUT.INT  MSS=3

PARAMETERS:

INP

 1. Input dataset (MSS format image or IBIS tabular file) 2. Optional mask image for 'TOIBIS mode.

OUT

 Output dataset (IBIS tabular file or MSS format image)

SIZE

 Standard VICAR size field. Refers to one band in MSS format image.

SL

 Starting line Not used in 'TOMSS.

SS

 Starting sample Not used in 'TOMSS.

NL

 Number of lines

NS

 Number of samples in one band

MODE

 Keyword for direction ('TOIBIS or 'TOMSS)

MSS

 Number of bands in MSS format dataset.

BANDS

 Which bands to use in MSS format image.

COLS

 Columns for data in IBIS file.

NCOL

 Number of columns in output IBIS file.

INC

 The increment (or subsampling) factor for 'TOIBIS mode.

FORMAT

 The pixel format for the output MSS format image.

UPDATE

 Indicates the OUT file is an old file to 'UPDATE.
