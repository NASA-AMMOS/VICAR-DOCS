jpl.mipl.mica.util

Class ImageChain

  • java.lang.Object
    • jpl.mipl.mica.util.ImageChain

  • public class ImageChain
extends java.lang.Object
    A utility class used to maintain a piece of a Java Advanced Imaging chain. Operations can be added and removed from the chain in a dynamic fashion and the ImageChain will preserve the JAI rendering dependency heirarchy.

    Rendered operations will be generated and managed automatically. All that is provided is the operation ParameterBlock and the operation name. A key value is used to retrieve, modify, or delete the operation from the list at a later time. Adding RenderedOperators directly is not possible because a chain must be able to dynamically link and unlink source and sink operators. Only a fixed base image can be specified.

    NOTE: The chain should only be used for renderable operations, not rendered. Rendered chains can not be modified after they are rendered, so a dynamically modifiable chain is not possible.

    NOTE: Additions or deletions from the chain will likely affect the current top image operator. If this chain is used as a source for other operations, the link to the top of this chain will need to be revalidated.

    • Method Detail

      • addOp

        public void addOp(java.lang.String keyname,
                  java.lang.String optype,
                  java.awt.image.renderable.ParameterBlock pb)
        Adds or modifies an operator. If an operator does not already exist with the given key name, the ParameterBlock is registered and a new RenderedOp is linked onto the top of the ImageChain. If an operator has already been added with the given key name, the operator's ParameterBlock is replaced with the given ParameterBlock.

        Parameters:
        keyname - The String to be used as a registration key. Use this key again to modify or remove this operator.
        optype - The JAI operation name. Used to call JAI.createRendered(optype,pb).
        pb - The ParameterBlock to be used by the operation.

      • findOp

      • removeOp

      • getTop

        public javax.media.jai.RenderedOp getTop()
        Returns the top of the ImageChain. This operator should then be used to render the chain. It may also be used as the source to further rendering operations.

        NOTE: Additions or deletions from the chain will likely affect the current top image operator. If this chain is used as a source for other operations, the link to the top of this chain will need to be revalidated by calling getTop() again.
