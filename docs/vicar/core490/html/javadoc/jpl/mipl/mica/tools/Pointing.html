    public class Pointing
extends javax.swing.JPanel
implements java.beans.PropertyChangeListener
    A Pointing tool is a JavaBean that interacts with a set of ImageChips to correct errors in their pointing parameters. A tool operates on all ImageChips in the current selection, displaying the pointing parameters of the first ImageChip in its display. A tool adapts its interface to the number and the types of parameters that the chip PointingModels have. For PointingModels that have azimuth and elevation or right ascension and declination parameter pairs, a Pointing tool provides a special "nudge" feature. At a minimum, for all parameters the name and value are displayed and can be edited directly.

    There is also a feature to re-point a camera using the mouse. When the check box on a Pointing tool is set, the mouse can be used to "drag & point" the first selected chip's camera. Vertical mouse movement will change the elevation or declination. Horizontal mouse movement will change the azimuth or right ascension. The apparent pixel motion of the image chip may not track with the pixel motion of the mouse because the origin of the view may not correspond with the chip's camera origin.

    All these features are provided through a plug-in architecture. A Pointing tool inspects the ImageChips in the current selection, then configures itself using objects implementing the PointingPlugin interface. The PointingNudge and PointingList are two such PointingPlugins that the tool uses. For mouse control, a Pointing tool creates a PointingOverlay and inserts that into a global Display object. For details on how these objects function, refer to their individual documentation.

      • setSelection

        public void setSelection(java.util.Vector selection)
        Takes a newly set selection, inspects the ImageChips, (un)loads PointingPlugins, then initializes them with the new selection's pointing data. Calls the addPlugin and syncPlugins methods to ensure that the correct plugins are loaded but that no extra work is done creating/destroying plugins if they're already loaded. The configurePlugin method is called to set the plugin's values.

      • addPlugin

        protected PointingPlugin addPlugin(PointingPlugin plugin,
                                   java.util.Vector oldPluginList,
                                   java.util.Vector newPluginList)
        This method adds a new PointingPlugin to the plugin panel. If there is already a plugin of the same type already being displayed, then the existing one is used. The method returns the value of the plugin that is actually being displayed.
        Parameters:
        plugin - A plugin to display. Not used if another of the same type is available.
        oldPluginList - The list of plugins that was in the plugin panel prior to this regeneration.
        newPluginList - The (possibly empty) new list of plugins being assembled for the new ImageChips.
