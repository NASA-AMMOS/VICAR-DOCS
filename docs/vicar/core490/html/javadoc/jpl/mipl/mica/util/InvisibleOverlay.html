    public abstract class InvisibleOverlay
extends javax.swing.JComponent
implements java.awt.event.MouseListener, java.awt.event.MouseMotionListener, java.awt.event.ActionListener
    The InvisibleOverlay is used by a tool to add functionality to the jpl.mipl.mica.Display object. An overlay may be inserted into the Display inside the Display's JScrollPane and on top of the mosaic drawn there. The Display passes through events to the overlays that register with it and allows them to appear to draw onto the mosaic.

    By default, an overlay will be granted access to the mouse events in the Display. Only the "top" overlay will receive these events, blocking others until the top overlay is removed. If an overlay does not wish to listen for events, it should set its Activate flag to false so that it will not prevent other overlays from receiving events.

    To draw on the overlay, a subclass should override the paintComponent() method.
