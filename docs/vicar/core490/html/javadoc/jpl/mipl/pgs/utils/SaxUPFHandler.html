Class SaxUPFHandler

    public class SaxUPFHandler
extends org.xml.sax.helpers.DefaultHandler
    XML parser handler for the PGS UPF/Configuration parsing. This class constructs a map with keys equals to the fully qualified name of the XML element. The value of each key is a string. For example
    
   
     
       somedir
    is stored in map with key=pgs.upf.logDir and value=somedir. This class allows variable to be used in defining value of a key as follows: 1) ${ENV.var} maps the var to Operating System defined environment variable only. 2) ${UPF.var} maps the var to a variable defined in UPF. For example,
    now the use of ${UPF.WORKING_DIR} will map to /project/rt/dev/rrp/pgs/test. 3) ${var} first trying to find var in the environment variable defined in the UPF using construct, if not found, it checks the Java Runtime Environment variables (System.getProperty(var)) and if it is not defined, it checks the OS environemnt variables.
