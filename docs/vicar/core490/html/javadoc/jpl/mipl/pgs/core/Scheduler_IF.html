Interface Scheduler_IF

    All Known Subinterfaces:
    EventScheduler_IF, MarsScheduler_IF
    All Known Implementing Classes:
    EventScheduler, MarsScheduler

    public interface Scheduler_IF
extends java.lang.Runnable
    The scheduler interface that is expected by the controller. The controller is responsible for starting and stopping the scheduler. The scheduler uses Controlloer_IF::releaseRequest method to run a request (i.e., a pipeline). The following key/value can be passed in request map to control some aspects of the pipeline:
    • SFID (optional) specify the ServiceFactory ID that should handle this request. An exception is thrown if the factory is down or not accepting any request.
    • logger (required) used by JBPM execution engine framework to log messages. See ProxyLogger implementation for an example.
