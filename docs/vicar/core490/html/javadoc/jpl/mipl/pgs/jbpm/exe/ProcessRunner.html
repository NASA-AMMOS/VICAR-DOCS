    public class ProcessRunner
extends ProcessController
implements java.lang.Runnable, ProcessRunner_IF
    Executes the given process definition or process instance. Depending on the parameters in the UPF, this class either resumes a process instance saved in database or creates a new process instance using the process definition specified in the UPF or a saved process definition from the database. Once the process instance is available, it is decorated with appropriate action handlers and all the parameters from the UPF are added to the process instances' execution context.

        Create a process instence dependig on UPF parameters. The priority is as follows: If instanceID is specified it is used first. If not and if process definition name is specified it is used. If neither one of them is specified then an instance is created using the process definition specified by UPF key word UPF.PROCESS_DEFINITION_FILE_NAME
        Parameters:
        upf - the user specific parameters. The key should all be String with values as defined in UPF class.
