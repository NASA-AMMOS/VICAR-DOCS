Interface ProcessRunner_IF

    ProcessControl_IF, java.rmi.Remote, java.lang.Runnable
    All Known Implementing Classes:
    ProcessRunner

    public interface ProcessRunner_IF
extends ProcessControl_IF, java.lang.Runnable
    Executes the given process definition or process instance. Depending on the parameters in the UPF, this class either resumes a process instance saved in database or creates a new process instance using the process definition specified in the UPF or a saved process definition from the database. Once the process instance is available, it is decorated with appropriate action handlers and all the parameters from the UPF are added to the process instances' execution context.
