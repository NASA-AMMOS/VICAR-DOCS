    public class DispatchableMissionSpecInstrumentInfo
extends java.lang.Object
implements MissionSpecInstrumentInfo
    Purpose: A collection composite that combines the Mission spec instrument info instances from multiple file finders as one. A set of instrument names and categories will be created such that names and categories returned from underlying instances. Translation methods are implemented in such a way that the first underlying instance which returned a non-null value is the value that will be returned.
