Package jpl.mipl.mars.mis.unpackers

  • Class Summary 
    Class Description
    AbstractUnpacker
    Abstract super class extended by all Unpackers that will be used to extract band data from an added image.
    ArmPreloadUnpacker
    An extension of AbstractUnpacker designed to take care of ArmPreload RDR DN value mapping.
    ArmReachabilityUnpacker
    An extension of AbstractUnpacker designed to take care of ArmReachability RDR DN value mapping.
    DefaultUnpacker
    An extension of AbstractUnpacker that provides a best guess mapping when no image type is available or the image type provided is unkown.
    ReachabilityUnpacker
    An extension of AbstractUnpacker designed to take care of Reachability RDR DN value mapping.
    StandardLookupUnpacker
    Extension of the StandardUnpacker which adds the ability to map values to strings.
    StandardUnpacker
    This is the goto unpacker for any image type that only requires variation on key names by the number of bands (and no extra processing).
    Unpacker
    This is a static method "Toolbox" class that aims to allow unpacking of sample data into meaningful values that are linked to the proper names.
