      • refresh

        protected void refresh()
        Refreshes this panel according to the number of components and state. There are currently four possibilities: (1) No components to display, tabbed pane need not be viewed. (2) No components to display, but tabbed pane should be viewed. (3) One component to display, but without tabbed pane. (4) Show all components in tabbed pane
