    public abstract class AbstractDispatchableImageFileFinder
extends AbstractMarsImageFileFinder
implements DispatchableMarsImageFileFinder
    Purpose: A collection composite that combines multiple file finder instances as one. There are some constraints upon the member file finders that must be met: 1) They must share the same file finder root. 2) Their EDR regex must be unique and unambiguous. Constraint 1. ensures that the case of setting of a composite acts similar to the case of a single file finder. Constraint 2. Requests are dispatched to a file finder by checking with each finder's isProductNameFormatted() method. The first match is used. To ensure that requests are dispatched correctly, the method should use unambiguous criteria across member finders. The default implementation compares the string argument with the finder's EDR regular expression.
