Class ArmReachabilityUnpacker

  • java.lang.Object

      • jpl.mipl.mars.mis.unpackers.ArmReachabilityUnpacker

  • public class ArmReachabilityUnpacker
extends AbstractUnpacker
    An extension of AbstractUnpacker designed to take care of ArmReachability RDR DN value mapping. Unlike many other implementations of AbstractUnpacker, this class adds header data to the map it returns to aid the end user. The return values are exracted from the int array and are pulled as 2bit numbers packed into the first 16 bits of each sample.
