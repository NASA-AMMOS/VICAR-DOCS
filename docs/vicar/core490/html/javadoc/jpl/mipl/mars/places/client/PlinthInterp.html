Class PlinthInterp

  • java.lang.Object
    • jpl.mipl.mars.places.client.PlinthInterp

  • public class PlinthInterp
extends java.lang.Object
    Interpolate a point for Plinth. This is not a simple linear interpolation but rather treats the traverse kind of like a rubber sheet, doing a rotation and scale rather than just a scale (which would be simple linear interpolation).

    The constructor sets up what the interpolation should be; then transformPoint can be called for each point on the path.
