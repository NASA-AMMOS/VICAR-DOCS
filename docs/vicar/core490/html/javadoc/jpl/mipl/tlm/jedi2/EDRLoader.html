Class EDRLoader

  • java.lang.Object
    • jpl.mipl.tlm.jedi2.EDRLoader
  • All Implemented Interfaces:
    EDRReader

    public class EDRLoader
extends java.lang.Object
implements EDRReader
    This class determines what type of file the target EDR is (VICAR, ISIS, FITS, ...) and loads the data accordingly. Objects that use EDRLoader for this purpose do not need to concern themselves with the filetype, though they do need to check for a FormatNotSupportedException. (which I haven't implemented yet :-/ )
