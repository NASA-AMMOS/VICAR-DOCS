    public class Type02CHDO
extends GenericCHDO
    CHDO for secondary CHDO, type 02. Typically, this CHDO is used to obtain the ERT of the packet. This CHDO contains the following info (H-channel designations are for SIRTF): 
       |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
 0 |                           chdo_type                           |
   |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
 2 |                          chdo_length                          |
   |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
 4 |                                                               |
   |--                record_id                                  --|
 6 |                                                               |
   |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
 8 

   where record_id is:

   |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
 0 |           major               |           minor               |
   |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
 2 |         mission_id            |           format              |
   |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
