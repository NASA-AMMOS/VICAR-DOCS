    ConsumerStreamEventInterface

    public class DisplayServer
extends ConsumerStreamEventAdapter
    This is the JEDI Server which is run as either a standalone daemon, or a ProductAvailabelEventListener in the new TPS CORBA world. Eventually, I'd like to make this a Servlet to make things better/easier. There isn't really all that much to running this thing. Simply supply a port number (default: 20202).
