      • setBands

        public void setBands(int[] bandIndices)
        Sets bands according to bandIndices parameter. If parameter is of length 1, then all bands will be set to that value. If length 2, then band 3 will be set to the latest value. If length is greater than three, only the first three are used.
        Parameters:
        bandIndices - Array of bands to select
        Throws:
        java.lang.IllegalArgumentException - if bandIndices is either null or of length 0.
