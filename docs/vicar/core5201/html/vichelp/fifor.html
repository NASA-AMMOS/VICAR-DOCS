Help for FIFOR

PURPOSE

     FIFOR reads the value parameter from the named pipe

TAE COMMAND LINE FORMAT

     local somestrvar string
     fifor pipename=thePipeName strvar=somestrvar

OPERATION

     FIFOR reads the value parameter from the named pipe and copies it
     to strvar.

     
RESTRICTIONS

     The named pipe must exist and be readable. This program will block
     until there is a value to read.
