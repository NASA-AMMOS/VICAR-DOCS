Help for WV2REF

PURPOSE
 This program converts raw dn images
 into top of atmosphere reflectance 
 images where the values range from 
 0 to 1.  The output image(s) is in 
 "REAL" format. 

REVISIONS
2022-08-10 B. Crocco afids to opensource (untested)

PARAMETERS:

INP

 Input raw dn worldview2 image(s). The number of inputs must be either 1 or all 9 bands. !!! PAN BAND IS 9 !!!

OUT

 Output reflectance image(s). Either 1 or all 9 bands depending on the input !!! PAN BAND IS 9 !!!

MULTIMETA

 .IMD file containing metadata regarding the multispectral input images.

PANMETA

 .IMD file containing metadata regarding the pan input image

BAND

 Specifies which band the input image is. Ignored if all 9 bands are input. !!! PAN BAND IS 9 !!! Coastal=1 Blue=2 Green=3 Yellow=4 Red=5 Red Edge=6 NIR1=7 NIR2=8 Pan=9 .END
