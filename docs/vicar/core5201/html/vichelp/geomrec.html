Help for GEOMREC

TRANSFORMS RANGE MEASUREMENT TO THE DISTANCE ON THE SURFACE.
(TRANSFORMS FROM SLANT RANGE TO GROUND RANGE)

COGNIZANT PROGRAMMER:  JAN HEYADA

PARAMETERS:

INP

 input dataset

OUT

 output dataset

SIZE

 standard VICAR size field

SL

 starting line

SS

 starting sample

NL

 number of lines

NS

 number of samples

HEIGHT

 sensor height (meters) specified as an integer

RSSPACE

 range sample spacing (microns) specified as an integer

ASSPACE

 azimuth sample spacing (microns) specified as an integer

RSCALE

 range scale (meters/micron on film) specified as a floating point

ASCALE

 azimuth scale (meters/micron on file) specified as a floating point

THETA

 angle from vertical of first return (degrees) specified as a floating point
