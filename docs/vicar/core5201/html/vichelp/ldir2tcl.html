Help for LDIR2TCL

PURPOSE:
ldir2tcl copies the local directory path to a passed string variable

EXECUTION:

Example:

ldir2tcl strvar=ldir
            where ldir is a TAE TCL variable of type string
          
TIMING: 

Instantaneous.
