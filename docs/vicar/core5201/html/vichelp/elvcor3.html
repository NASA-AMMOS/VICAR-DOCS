PURPOSE
     Calculate elevation offsets for a satellite image using a
DTED elevation image and satellite az-elev
	   
CALL
     
OPERATION


PERFORMANCE

Restrictions
------------
The dted must cover the area and have a GeoTIFF label.

PARAMETERS:

GRIDFILE

 ibis file with (line,samp) in cols (1,2) referencing gridimage

GRIDIMAGE

 A GeoTIFF labelled image used to map the (line,samp)

DTED

 elevation file with GeoTIFF label covering area, in m.

SATAZIM

 satellite azimuth, clockwise from North in degrees

SATELEV

 satellite elevation, nadir is ninety degrees .REGELV registration elevation from proc .COLS Generated and corrected pixels in these columns (oldline,oldsamp, newline,newsamp) .END
