Help for PAINTC

PURPOSE
This program recolors the dn in the input
image with the corresponding color in the
colormap ibis image.  The columns of the
colormap must be specified or defaults to
(1, 2).
If an inp dn is not specified in the
colormap, the out dn will default to the
bdn (background dn).

PARAMETERS:

INP

 Input image file Input colormap IBIS file

OUT

 Output image file

COLS

 Columns of the colormap 1st column is input dn 2nd column is output dn

BDN

 Background color .END
