Help for LINSHFT

PURPOSE:

Calculates the shift to be applied to file 2 along the correlation line
to obtain agreement with file1.

Subpixel not implemented yet.

EXECUTION:

alz please do

 TIMING: 

less than one second

PARAMETERS:

INP

 two images for matching

ITIE

 two points in image 1; defines a line for match

OTIE

 two approximately matching points in image 2

XDELTA

 two increments for cross track correlation (help 2)

XNUM

 number of cross track pts

SHIFTMAX

 limit search for shift; + or - this value

SHIFT

 return variable for linear shift along line (real)

XSHIFT

 return variable for linear shift across line (real)

CORVAL

 return variable for correlation value (real)

SUBPIX

 perform subpixel shift

APODIZE

 taper end values to avoid end effect

ZTHRESH

 inclusive range for pixel values to correlate .LEVEL 2

XDELTA

 The first value is added to the line coordinate of the otie and otie and the second value is added to the sample coordinate. The user should make sure that the xdelta is in a perpendicular direction. .END
