PARAMETERS:

INP

 Image with ephemeris, 2d file is IBIS file, 5+ cols

ACS_TIME

 ACS time in seconds since 2000-01-01T12:00:00 UTC.

COLS

 Columns to receive the (samp,line,RA,DEC) RA and DEC in decimal degrees

URANGE

 Start_samp, step, end_samp

VRANGE

 Start_line, step, end_line

HRANGE

 Start_elev, step, end_elev

TOD_T_SV

 Position vector of s/c with respect to True-of-Date (TOD) coordinate frame

TOD_Q_SV

 Orientation quaternion for s/c with respect to True-of-Date (TOD) coordinate frame

SV_T_C

 Position vector of camera with respect to s/c (SV) coordinate frame

SV_Q_C

 Orientation quaternion for camera with respect to s/c (SV) coordinate frame

FU

 Focal param in sample dir

FV

 Focal param in line dir

Q

 Pixel skew

U0

 Sample coord of principal point

V0

 Line coord of principal

KAPPA

 Lens distortion params

CALC_MODE

 0 = earth first, if off earth, then space 1 = earth only 2 = space only, also earth transparent .LEVEL1

HRANGE

 Spheriod, Geoid, and Terrain elevation handled by the VICAR proc, ignored for space looking case .END
