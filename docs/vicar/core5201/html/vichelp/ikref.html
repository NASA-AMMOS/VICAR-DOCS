Help for IKREF

PURPOSE
 This program converts raw dn images
 into top of atmosphere reflectance 
 images where the values range from 
 0 to 1.  The output image(s) is in 
 "REAL" format.  Band 5 is the pan
 band.  The order is blue, green,
 red, nir, and pan.

REVISIONS
2022-08-10 B. Crocco afids to opensource (untested)

PARAMETERS:

INP

 Input raw dn ikonos image(s). The number of inputs must be either 1 or all 5 bands. Order is blue, green, red, nir, pan.

OUT

 Output reflectance image(s). Either 1 or all 5 bands depending on the input

META

 .IMD file containing metadata regarding the multispectral input images.

BAND

 Specifies which band the input image is. Ignored if all 5 bands are input. .END
