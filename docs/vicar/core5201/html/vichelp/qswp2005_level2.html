Level 2 Help for QSWP2005

INP

 
STRING - Input fft file - maybe any type of file

OUT

STRING - Output file - has the same format as the input file

OP

STRING - Specifies forward or reverse swapping.  If "FORWARD",
         this tells the program that the quadrants in the input
         dataset HAS NOT been swapped.  If "REVERSE", this
         tells the program that the quadrants in the input
         dataset HAS been swapped.
