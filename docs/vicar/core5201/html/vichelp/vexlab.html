Help for VEXLAB

PURPOSE

     VEXLAB reads the metadata file generated by vextract and adds as GeoTIFF labels

TAE COMMAND LINE FORMAT

     vexlab INP=A META=B
     
     where

     A           a VICAR image
     B           a metadata file from vextract
