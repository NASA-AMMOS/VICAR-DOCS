Help for RPC2SC

PURPOSE:

The spacecraft ephemeris is put into 8 columns of an ibis file 
(position quaternion followed by attatude quaternion).

The second record contains the variables:
m,q,tau,theta,phi,psi,bu,bv in that order.

PARAMETERS:

INP

 File containing RPC, can be GeoTIFF image or IBIS file formatted

OUT

 8 column IBIS file with the spacecraft quaternions in WPP and a second record containing calculated variables (see main document).

COLS

 Columns if IBIS file format

LONRANGE

 Longitude limits of volume of interest

LATRANGE

 Latitude limits of volume of interest

HRANGE

 Elevation limits of volume of interest

U0

 Sample coord of principal point

V0

 Line coord of principal point

ALTITUDE

 altitude of spacecraft .END
