Package jpl.mipl.util Description

Miscellaneous MIPL utilities.

The package includes the following general capabilities:

  • Amoeba function minimization algorithm.
  • The Java Query Interface (jqi) database utilities.
  • A workaround for a bug in JDK 1.3's ComponentColorModel./li>
  • A utility class for accessing Service Provider Interface (SPI) files from a jar.
