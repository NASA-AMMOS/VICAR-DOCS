        public PathFinder()
        Constructs the PathFinder from the FFPaths.xml. Looks first in WEB_INF/FFPaths.xml then the file located in this directory and loads first one with data into a static Map for lookup by the next level (it is cached). NOTE: it will not be refreshed if the xml is changed after deployment
