        public java.lang.String getSourceProductPath(java.lang.String rdrPath)
                                      throws FileFinderException
        Returns the path of the source EDR for nominal RDR products, or the path of the linearized version of that EDR for linearized RDR products. NOTE: Behavior of this method must be constrained such that a 'source image' will return the path of itself.
