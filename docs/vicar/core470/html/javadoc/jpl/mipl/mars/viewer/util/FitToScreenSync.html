Class FitToScreenSync

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.FitToScreenSync

  • public class FitToScreenSync
extends java.lang.Object
    Purpose: Singleton that synchronizes with parts of the application model so that it can determine zoom for the current application state. This is mostly needed for zoom modes that attempt to fit the current image in the displayable area.
