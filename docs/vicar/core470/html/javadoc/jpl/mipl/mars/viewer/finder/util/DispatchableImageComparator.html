Class DispatchableImageComparator

  • java.lang.Object
    • jpl.mipl.mars.viewer.finder.util.DispatchableImageComparator
  • All Implemented Interfaces:
    java.util.Comparator

    public class DispatchableImageComparator
extends java.lang.Object
implements java.util.Comparator
    Purpose: Accepts an instance of DispatchableMarsImageFileFinder and forwards requests to underlying file finders based on the product filename format. This implementation compares the underlying file finders of the two objects being compared. If those finders are the same, then that finder is used for comparison. Otherwise, a default comparator will be used.
