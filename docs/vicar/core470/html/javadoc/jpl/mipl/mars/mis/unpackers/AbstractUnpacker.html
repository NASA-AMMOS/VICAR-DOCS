      • parse

        public abstract java.util.Map<java.lang.String,java.lang.Object> parse(float[] fvals,
                                                                       int[] ivals)
        Uses the implementation specific setup to extract the band data from the float[] and int[] samples passed in. For some implementations this method may be inflating (|Map| > |fvals|). It is up to the implementation what the return type(Object) is although it is usually a String, Integer or a Float some implemenations use float[] some use int[] so ensure |fvals| = |ivals|
        Parameters:
        fvals - a float[] taken from the raster as a float sample
        ivals - an int[] taken from the raster as an int sample
        Returns:
        Map<String,Object> a map from band key to band value.
