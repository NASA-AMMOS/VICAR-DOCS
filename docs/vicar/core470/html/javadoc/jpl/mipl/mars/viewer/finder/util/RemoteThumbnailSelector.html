Class RemoteThumbnailSelector

  • java.lang.Object
    • jpl.mipl.mars.viewer.finder.util.RemoteThumbnailSelector
  • All Implemented Interfaces:
    ThumbnailSelector

    public class RemoteThumbnailSelector
extends java.lang.Object
implements ThumbnailSelector
    Purpose: Implementation of the ThumbnailSelector interface used for remote file finders. A cache of results is maintained to speed up repeated queries. Implementation takes two parameters, the first is the product for which a thumbnail is requested, and the second is a list of potential thumbnails. This list is examined and the first match (based on group id) is returned as the thumbnail product.
