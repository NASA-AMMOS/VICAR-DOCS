      • getThumbnail

        public java.lang.String getThumbnail(java.lang.String imagePath)
        Returns the best match thumbnail for a given image product path. A more specific lookup is performed using product metadata, followed by a more general lookup using group id only. Finally, if still no match, we look to see if product itself is a thumbnail and return it if so.
        Parameters:
        imagePath - Path to image product
        Returns:
        Path to thumbnail if found, null otherwise
