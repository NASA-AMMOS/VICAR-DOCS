    public class RdrContourStretchImageContent
extends RdrImageContentBase
    Purpose: Encapulates the JAI render graph and controls for an RDR file which uses the Contour Stretch operator for viewing (eg. XYZ, Range) Includes the ContourStretchOp operator at end of render graph, and the associated control panel.
