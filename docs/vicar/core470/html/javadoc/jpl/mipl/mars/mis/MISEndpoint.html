Class MISEndpoint

  • java.lang.Object
    • javax.websocket.Endpoint
      • jpl.mipl.mars.mis.MISEndpoint

  • public class MISEndpoint
extends javax.websocket.Endpoint
    This class provides a concrete object for instantiation by the server for each connection requested by a client. By the default implemented in the Config a separate instance is created for each incomming connection ensuring a one to one correspondance between open sockets and MISEnpoint instancs.
