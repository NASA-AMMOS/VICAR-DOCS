    public class MslOpgsIv2ObjSimpleService
extends BaseAntillesService
    Purpose: Implementation of the ANTILLES service using a Linux script that accepts the location of a IV file and an output directory and creates a single model and texture representing a unified model across all tiles with lowest LOD (least detailed). Documentation of script: # Make an OBJ file (and associated materials and skin files) from a # mesh file. Copies the mesh to the output directory as a gzip'd tar file # containing the OBJ, MTL, and PNG files. # # Usage: # make_obj_server.csh input.iv output_dir # # The input file should be a fully qualified pathname (with $ODS_SOL/...). # A working dir under the current dir is created with the input basename # with _tmp appended.
