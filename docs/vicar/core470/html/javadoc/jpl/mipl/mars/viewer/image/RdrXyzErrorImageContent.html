    public class RdrXyzErrorImageContent
extends RdrErrorImageContent
    Purpose: Encapulates the JAI render graph and controls for an ERROR RDR file. Includes the rescale and format operator at end of render graph, and the associated control panel. Default data range = (-1.0, 1.0).
