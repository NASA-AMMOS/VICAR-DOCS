        protected int determineNewEdrIndexToSelect(java.util.List newEntries,
                                           java.lang.Object selected)
        Convenience method that will determine the index of the EDR list to be selected given a list of new entries and the value of a current entry.         selected - Instance of current selected
        Returns:
