jpl.mipl.pgs.jbpm.services

Class ContextService

  • java.lang.Object
    • jpl.mipl.pgs.jbpm.services.ContextService

  • public class ContextService
extends java.lang.Object
    This is a utility class that is used by jpl.mipl.pgs.jbpm package for adding/removing context variables. This is a thread safe class that allows manipulation of process instance context. The context contains variables that are passed from node to node.
