      • SERVER_ADMIN

        public static final UserPermission SERVER_ADMIN
        Permission masks Admin Permissions - these permissions are awarded for the whole mission. (applies to all the instruments) "ADMIN" - has permissions such as add, modify, delete, etc. IMPORTANT: Make sure to also grant the INQUIRY permission whenever the ADMIN permission is granted. The perstore does NOT implicitly give INQUIRY if ADMIN is given.
