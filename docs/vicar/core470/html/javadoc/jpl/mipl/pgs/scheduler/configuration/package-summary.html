Package jpl.mipl.pgs.scheduler.configuration Description

Who
 When
 What
Raj Patel
 09/30/2011
 Initial Release




This package contains MATIS Command and Configuration classes for parsing various configurations.

The default ready to use implementation provides a class for parsing one particular type of configuation. In this sense, there is a one-to-one correspondence between a XSD file and a configuration class. Each configuration class parses and stores known fileds. Any unknown filed is stored as generic field that can be accessed via the generic API provided by the class. As so, there is no need to write a configuration class for any enahancement to existing XSD files.
