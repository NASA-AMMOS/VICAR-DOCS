Package jpl.mipl.pgs.utils.logging Description

PGS Logging Utilities Package

This package contains the utitlies for handling logging in the P roduct G eneration S ubsystem (PGS). It is an implementation of the java.util.logging.Logger class. It gives control to the calling class on how to format the messages. It allows logging to a specified file and also allows archiving of log files as they reach the specified size limit.
