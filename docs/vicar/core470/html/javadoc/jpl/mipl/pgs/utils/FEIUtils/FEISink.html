    Wraps FEI product delivery APIs. The class is intended to be used by Worker_Object and thus wraps the FEI product delivery API's only. This class by default uses SSL as FEI's security model and domain file as specified by FEI5.

      • FEISink

        public FEISink(FEIErrorHandler errorHandler,
               java.util.logging.Logger logger,
               UPF upf)
        throws SessionException,
               MissingParameterException
        Parameters:
        errorHandler - for handling FEI related errors.
        logger - for logging
        upf - Must contain the following keys
        • fei.username: The user name to be used to connect to the server.
        • fei.password: The password to be used to connect to the server.

        The properties may also contain the following parameters:
        • fei.domain: The FEI domain file to be used. The default is $FEI5/domain.fei
        • fei.keystore: The FEI key-store file to be used. The default is $FEI5/mdms-fei.keystore
        • fei.securitymodel: The security model that is to be used. Default is to use SSL.

        Throws:
        SessionException - if the session throws an exception MissingParameterException if FEI5 environment variable is not defined.
        MissingParameterException
