    public class UPF
extends java.lang.Object

    Allows parsing of User Preference File. The UPF object allows users to specify simple parameters such as 
    
 
   
     somedir
    to more complex and dynamic parameters such as using the <script></script> element.
    Refer to UPF Users Guide for more information.
