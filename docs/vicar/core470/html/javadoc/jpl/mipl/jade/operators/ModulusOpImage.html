    public class ModulusOpImage
extends javax.media.jai.PointOpImage
    Purpose: A JAI operator which performs modulus function to each pixel with a bounded range. The range forms a circular sequence of values, so the rangeMax+1 = rangeMin and rangeMin-1 = rangeMax. A ModulusOpImage should not be instantiated directly, but through the JAI operation registry.
