jpl.mipl.mdms.FileService.komodo.registry.handlers

Class FileListHandlerFactory

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.registry.handlers.FileListHandlerFactory

  • public class FileListHandlerFactory
extends java.lang.Object
    Purpose: A simple factory that encapsulates the creation of FileListHandler instances and format messages. Constructor requires an API version which is referenced during creation calls. In general, the client will invoke createMessageFormat(), and pass in the result to one of the createHandler() methods.
