Class ExtensionFactory

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.ui.savannah.subscription.util.ExtensionFactory

  • public class ExtensionFactory
extends java.lang.Object
    Purpose: Factory that builds instances of configuration info based on the extension point attribute of an extension Element. A lookup is performed to ascertain the class to be used for a particular extension. This information must be loaded from an extension info properties file with extension point as key and info class as the value. A default extensions info file is provided. However, it can be overriden by specifying the new extensions info file as the value of the komodo.extensions.info property.
