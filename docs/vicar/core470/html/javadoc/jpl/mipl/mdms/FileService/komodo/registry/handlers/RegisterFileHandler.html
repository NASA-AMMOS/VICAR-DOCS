    RegistryHandler

    public class RegisterFileHandler
extends GeneralErrorReportingHandler
    Implementation of the registry handler to handle register file request to the registry. All error messages are redirected back to the client. This class is derived from the GeneralErrorReportingHandler to take advantage of the automatic error capturing and reporting implementation.
