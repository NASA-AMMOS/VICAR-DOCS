jpl.mipl.mdms.FileService.komodo.registry

Interface Registry

  • All Known Implementing Classes:
    AbstractDBRegistry, MySQLRegistry, SybaseRegistry

    public interface Registry

    This interface defines the requirement for domain-specific implementation to be derived from. One example of such domain-specific implementation is to implement the registry using a relational DBMS. The definition does not suggest any exceptions to be thrown here, because it is expecting the domain-specific implementation to catch any errors and dispatch them through the RegistryHandler objects.
