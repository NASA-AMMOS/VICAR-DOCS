    public class NewMetaSubscriptionPanel
extends javax.swing.JPanel
    Purpose: JPanel for entering the meta-subscription type, filetype, and output directory. Subscription types and filetype options are passed as a string array, while the initial output dir is passed as a file. Once the panel is dismissed, query the methods get*() to retrieve selected parameters.
