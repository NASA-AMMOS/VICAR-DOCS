Class SocketAcceptor

  • java.lang.Object
    • jpl.mipl.mdms.connection.SocketAcceptor
  • All Implemented Interfaces:
    java.lang.Runnable, ServiceHandler

    public class SocketAcceptor
extends java.lang.Object
implements ServiceHandler
    Implementation of the Acceptor design pattern [POSA2] for initializing new Service Handlers. This is a complete reimplemention of the original SocketAcceptor class due to New I/O support in JDK1.4 with multiplex non-blocking communication establishment. This implements the ServiceHandler interface that is required by the Reactor framework.
