Class DBConnSource

  • java.lang.Object
    • jpl.mipl.mdms.FileService.sql.DBConnSource

  • public class DBConnSource
extends java.lang.Object
    Get a data source for creating connections. Used to get connections for DBMS access from a data source. Connections may come from a pool, or may be created on-the fly, depending on the properties associated with the data source defined in the registry.
