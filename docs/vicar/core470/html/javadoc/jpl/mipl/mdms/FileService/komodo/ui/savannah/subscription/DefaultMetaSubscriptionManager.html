Class DefaultMetaSubscriptionManager

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.ui.savannah.subscription.DefaultMetaSubscriptionManager
  • All Implemented Interfaces:
    MetaSubscriptionManager

    public class DefaultMetaSubscriptionManager
extends java.lang.Object
implements MetaSubscriptionManager
    Purpose: Default manager class for FEI subscription and event scheduling. Maintains a hashtable of of subscription lists. Index is the subscription's source, using the Subscription.getSource() method.
