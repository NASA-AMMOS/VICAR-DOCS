        public QSearchService createService()
                             throws SessionException
        Creates a new instance of QSearchService. Examines the value of the property QConstants.PROP_SEARCH_SERVICE_IMPL for the implementation of the service to instantiate. If no value is found, then the default Atlas II service will be returned. Also, if there is a service location value for the property QConstants.PROP_SEARCH_SERVICE_LOCATION, then that value will be passed to the service before the service is returned.
