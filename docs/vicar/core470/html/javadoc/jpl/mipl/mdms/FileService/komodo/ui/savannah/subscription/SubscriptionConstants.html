      • wrapValue

        public static java.lang.Object wrapValue(java.lang.String keyword,
                                         java.lang.String value)
        Wraps value parameter in another object based on the type associated with the keyword. For example, if the keyword is KEY_CRC and value is "true", then Boolean.TRUE would be returned because CRC is a boolean option. If keyword was KEY_USERNAME and value "true", then the string "true" would be returned. Currently only Boolean and String are returned.
        Parameters:
        keyword - Option keyword
        value - Option value
        Returns:
        Wrapped object of type associated with keyword corresponding to value.

      • getNextStates

        public static int[] getNextStates(int currentState)
        Retrieve an array of legal next states given the current state of a metasubscription. If the current state is not recognized or an end state, then return array will be of zero length.

        Returns:
