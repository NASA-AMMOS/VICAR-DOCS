jpl.mipl.mdms.FileService.komodo.services.query.util

Class QueryClientUtil

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.services.query.util.QueryClientUtil

  • public class QueryClientUtil
extends java.lang.Object
    Purpose: Utility class that creates instances of QueryClient. Using the options and operation parameters passed into the constructor to build the login information and location of the query file. The query service location if determined by inspecting the value of the service location system property.
