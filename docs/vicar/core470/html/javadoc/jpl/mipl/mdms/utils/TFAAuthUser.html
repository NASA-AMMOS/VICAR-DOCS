Class TFAAuthUser

  • java.lang.Object
    • jpl.mipl.mdms.utils.TFAAuthUser

  • public class TFAAuthUser
extends java.lang.Object
    Utility class to perform RSA-TFA authentication. This uses the RSA API for performing 2 factor authentication. Configuration and auxiliary files needed: 1. sdconf.rec - RSA server admins supplies this file 2. securid - generated at the first time authentication is performed. Host specific 3. rsa_api.properties - configuration file that references previous 2 files.
