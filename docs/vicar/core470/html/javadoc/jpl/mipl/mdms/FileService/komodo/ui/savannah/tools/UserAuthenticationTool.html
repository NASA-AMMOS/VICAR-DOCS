Class UserAuthenticationTool

  • public class UserAuthenticationTool
extends java.lang.Object
    Purpose: This tool authenticates user credentials with the intended filetype. If unsuccessful, the tool queries the user for updated credentials. This loop can be set for a number of iterations before it returns indicating no success.

                                           java.lang.String fullfiletype,
                                           int maxAttempts)
                                    throws SessionException
        Attempts to authenticate user for a specific filetype. Will attempt at most maxAttempts times before aborting.
        Parameters:
        title - Title to be displayed for input panels
        username - Initial username Can be null.
        password - Initial password. Can be null.
        fullfiletype - Full filetype to which user is attempting to connect.
        maxAttempts - Maximum number of attempts to authenticate before aborting.
        Returns:
        String[] of length two where first entry is username, and second entry is password if successful. Null otherwise.
        Throws:
        SessionException
