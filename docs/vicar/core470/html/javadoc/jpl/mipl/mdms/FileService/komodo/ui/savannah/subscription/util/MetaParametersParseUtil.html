Class MetaParametersParseUtil

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.ui.savannah.subscription.util.MetaParametersParseUtil

  • public class MetaParametersParseUtil
extends java.lang.Object
    Purpose: Utility class that converts between maps of strings and maps of objects based on knowledge of type via property name. Conversion from string to object entails iterating over pairs and calling SubscriptionConstants.wrapValue() to instantiate the appropriate type. Conversion from object to string relies on the implementation of toString for the object.
