    SubscriptionHandler

    public class SavannahModelUpdateHandler
extends AbstractSubscriptionHandler
    Purpose: Handler that updates Savannah tranfer model and receival model with files that were retrieved during a subscription. Special attributes are: 
      (1) updateTransferModel - "true" if transfer model should be updated with new
                            file transfer, "false" otherwise, default true
  (2) updateReceivalModel - "true" if receival model should be updated with new
                            file transfer, "false" otherwise, default true
