      • isDirectory

        protected boolean isDirectory(java.lang.String entry)
        If entry is null or empty string, returns false. Otherwise, if not in a FEI filetype, we assume all entries are filetype directories, and thus return true. If in a FEI filetype, then the only directory is the parent directory, where we return true. Otherwise, return false.
