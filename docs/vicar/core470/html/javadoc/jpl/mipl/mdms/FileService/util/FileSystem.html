jpl.mipl.mdms.FileService.util

Interface FileSystem

  • All Known Implementing Classes:
    SimpleFileSystem, UnixFileSystem

    public interface FileSystem
    This is a generic interface for file system operations that are required by FEI. At the present time file system support is very limited by the current JDK (v1.4.1). This generic interface serves as generic file system wrapper, which will simplify support for non-Unix systems.
