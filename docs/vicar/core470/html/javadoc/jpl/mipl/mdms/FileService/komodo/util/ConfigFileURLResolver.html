jpl.mipl.mdms.FileService.komodo.util

Class ConfigFileURLResolver

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.util.ConfigFileURLResolver

  • public class ConfigFileURLResolver
extends java.lang.Object
    Purpose: Utility class that resolves the URL of the domain file. Caller can directly provide a path value, or it can be automatically retrieved from the system property. If a path is determined, this class will first check for a file with the given path. If found, the corresponding URL is returned. Otherwise, the path is considered a resource location, and the URL of the first matching resource will be returned.
