Class NullPlugin

  • java.lang.Object
    • jpl.mipl.mdms.utils.logging.NullPlugin
  • All Implemented Interfaces:
    LoggerPlugin

    public class NullPlugin
extends java.lang.Object
implements LoggerPlugin
    This is a null logger plugin class that implements the required plugin interface. It serves as a fallback plugin when the suggested plugin failed to load.
    Version:
