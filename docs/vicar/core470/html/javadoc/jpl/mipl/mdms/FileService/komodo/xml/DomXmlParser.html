jpl.mipl.mdms.FileService.komodo.xml

Class DomXmlParser

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.xml.DomXmlParser
  • Direct Known Subclasses:
    DbStmtsXMLParser, HelpXMLParser

    public abstract class DomXmlParser
extends java.lang.Object
    Abstract base class for DOM XML parser. Currently we are using Apache Xerces-J for our DOM parser, but the code should be portable enough to plug-in other parsers. NOTE: To use this parser class, please pass in the following System parameter: -Djavax.xml.parsers.DocumentBuilderFactory=org.apache.xerces.jaxp.DocumentBuilderFactoryImpl
