    SubscriptionHandler

    public class NotificationPanelHandler
extends AbstractSubscriptionHandler
    Purpose: Displays filenames in a dialog. Special attributes are: 
      (1) modal - "true" if dialog should be modal, "false" otherwise, 
               default false
  (2) title - Title to be displayed, default is "New Files Available"
