Class GeneralFileFilter

  • java.lang.Object
    • jpl.mipl.mdms.FileService.util.GeneralFileFilter
  • All Implemented Interfaces:
    java.io.FileFilter, java.io.FilenameFilter

    public class GeneralFileFilter
extends java.lang.Object
implements java.io.FileFilter, java.io.FilenameFilter
    Purpose: File filter for the local file system. Client supplies a filesystem wildcard expression, which is then used to filter files. Allows special treatment for directories using a directory flag.
