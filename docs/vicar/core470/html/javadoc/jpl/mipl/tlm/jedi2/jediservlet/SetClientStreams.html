Class SetClientStreams

    java.io.Serializable, javax.servlet.Servlet, javax.servlet.ServletConfig

    public class SetClientStreams
extends javax.servlet.http.HttpServlet
    Servlet that takes user input and does initialisation of client and other parameters. This servlet checks the roles available for the user, uses returned role to get the default personality file (or takes the user input URL for non-default personality file. Finds the allowable streams from the stream factory if TPS system is being used, or other streams running if standalone system is used (Cassini only). Also sets up a dummy stream to represent a directory if directory is selected. Initialises client object.

        Method called by http request. Checks if default personality is selected; if so, gets the correct URL from the LDAP server based on user-assigned roles. Checks if read from directory is selected; if so, assignes dummy stream to represent directory. If TPS system is selected, gets currently running streams from Stream factory, and checks those against user roles for allowed access. Uses this information to set up client object, and adds it to client manager. Calls the page containing the Jedi applet on completion of these tasks.
