Package jpl.mipl.msl.mdms.fei.handlers

Package containing classes for MDMS Komodo Client Handlers plugin for Remote MSL Replication.

See: Description

  • Interface Summary 
    Interface Description
    Configuration
    Purpose: Configuration property names and some default values.
    MSLFileDatabase
    Interface for MSL database classes.
    QueryResultHandler
    Purpose: Interface for query ResultSet handler that performs an action for each result set entry.
  • Class Summary 
    Class Description
    FileRecord
    Data structure represented a file record, including filename, full filetype, location, filesize and modification time.
    MslDerbyFileDatabase  
    MSLFileDatabaseFactory
    Factory instantiates instances of MSLFileDatabase based on the value of a designated property 'msl.database.implementation', which can be set in a passed-in Properties map or via the System properties.
    MSLFileDatabaseTool
    Purpose: Utility tool that offers some simple services in maintaining the embedded database.
    MslH2FileDatabase
    Purpose: Initial implementation of the MSLFileDatabase using the H2 database.
    MSLRemoteSubscriptionHandler
    The primary purpose of this handler is to ensure a limit on disk space usage for subscribed files on MSL remote locations.
    SystemInfo
    Purpose: System info is responsible for system level operations such as space metrics.
    SystemUtility
    Purpose: System utility is responsible for system level operations such as space metrics and file deletions.
