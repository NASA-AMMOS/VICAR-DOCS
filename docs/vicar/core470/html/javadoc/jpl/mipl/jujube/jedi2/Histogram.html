Class Histogram

  • java.lang.Object
    • jpl.mipl.jujube.jedi2.Histogram
  • All Implemented Interfaces:
    java.io.Serializable

    public class Histogram
extends java.lang.Object
implements java.io.Serializable
    This class embodies the elements needed to convey the basic statistics of an array of data. These elements are:
    • histogram (caveat below)
    • mean
    • standard deviation
    • median
    • 3rd and 4th moments
    • coefficients of skewness and kurtosis (not implemented yet)

    The Histogram model is stored in two arrays: one an int[], the other a double[] array. The first gives the total count of pixels at that DN (the index). The second gives the % of the population at that DN. One should remember that the first array element relates the histogram at the minimum DN (which may, or may not, be 0), with every DN after that accounted for.
