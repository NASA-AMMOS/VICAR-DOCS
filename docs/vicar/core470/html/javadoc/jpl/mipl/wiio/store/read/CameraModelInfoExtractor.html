jpl.mipl.wiio.store.read

Class CameraModelInfoExtractor

  • java.lang.Object
    • jpl.mipl.wiio.store.read.CameraModelInfoExtractor

  • public class CameraModelInfoExtractor
extends java.lang.Object
    Examines an instance of PigCameraModel and returns a set of attribute maps pertaining to that model. Currently supported are: 1) camera model parameters, which detail the type and settings of the camera; 2) pointing parameters, which include camera position and orientation.
