        public void setFormatToByte(boolean b)
        If this value is set to TRUE before fullRead() is called the images data will be converted to BYTE. This is a simple convenience to obtain an easily displayable BYTE image that should have a decent stretch applied.
        The extreme of the image will be taken and the image scaled correctly to fit to byte. If this flag is false the data will be returned in its native data type. The user can apply any JAI Operators to it.
