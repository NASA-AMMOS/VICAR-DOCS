jpl.mipl.io.vicar

Class LabelToBean

  • java.lang.Object
    • jpl.mipl.io.vicar.LabelToBean

  • public class LabelToBean
extends java.lang.Object
    This class provides static functions which transfer data between VicarLabelSet objects and a JavaBean object described by an array of LabelEntry's. The LabelEntry array describes the VICAR label item as well as the property to be set (or read) in the bean.

        public static void readLabel(VicarLabelSet set,
                             java.lang.Object bean,
                             LabelEntry[] entries,
                             java.util.List extraItems)
                      throws java.beans.IntrospectionException,
                             java.lang.reflect.InvocationTargetException,
                             java.lang.IllegalAccessException
        Translate from a VicarLabelSet to Bean properties. Items in the set which are not found in the LabelEntry table are added to the extraItems vector, if supplied.

        NOT IMPLEMENTED YET!!!!: elementNumber and "required" checks, and multiple keywords.

        Parameters:
        set - The VicarLabelSet containing the labels to read.
        bean - The JavaBean in which the properties will be set.
        entries - An array of LabelEntry objects describing the relationship between label items and bean properties
        extraItems - A List supplied by the caller. This function will add any unrecognized VicarLabelItem entries to this vector. May be null if you don't need this info. Note that an item is recognized if any of its elements are found in the table; the fact that element 2 might have been present but 1 and 3 aren't, is not tracked.
        Throws:
        java.beans.IntrospectionException - if the bean's BeanInfo can't be found
        java.lang.reflect.InvocationTargetException - if the set method is invalid.
        java.lang.IllegalAccessException - if the set method is invalid.
        java.lang.IllegalArgumentException - if the entry._type field is invalid.
        See Also:
        VicarLabelSet, LabelEntry

      • writeLabel

        public static void writeLabel(VicarLabelSet set,
                              java.lang.Object bean,
                              LabelEntry[] entries,
                              java.util.List extraItems)
                       throws java.beans.IntrospectionException,
                              java.lang.reflect.InvocationTargetException,
                              java.lang.IllegalAccessException
        Translate from Bean properties to a VicarLabelSet. Items in the extraItems vector are assumed to be VicarLabelItems as well, and are added to the Set.

        If a Valid flag property exists in the LabelEntry, then the value is only written out if the Valid flag is true, or if the item is required. NOT IMPLEMENTED YET!!!!: elementNumber and "required" checks.

        Parameters:
        set - The VicarLabelSet to be updated. Any existing items are left alone in case a name conflicts, in which case they are overridden.
        bean - The JavaBean from which to get the property values.
        entries - An array of LabelEntry objects describing the relationship between label items and bean properties
        extraItems - A List supplied by the caller. If present, items in this vector are assumed to be VicarLabelItems and are added to the end of the Set. May be null.
        Throws:
        java.beans.IntrospectionException - if the bean's BeanInfo can't be found
        java.lang.reflect.InvocationTargetException - if the set method is invalid.
        java.lang.IllegalAccessException - if the set method is invalid.
        java.lang.IllegalArgumentException - if the entry._type field is invalid.
        See Also:
        VicarLabelSet, LabelEntry
