Class PDSCodec

  • java.lang.Object
    • com.sun.media.jai.codec.ImageCodec
      • jpl.mipl.io.codec.PDSCodec

  • public final class PDSCodec
extends com.sun.media.jai.codec.ImageCodec
    A subclass of JAI ImageCodec that handles the PDS image.

    Currently the codec defaults to the simplest usual cases, which are:
    1 band is a grayscale image
    3 bands are an RGB image.
    Future enhancements will includePDS specific extentions of com.sun.media.jai.codec.ImageDecodeParam and com.sun.media.jai.codec.ImageEncodeParam
    The encode and decode params will allow the user to tell the codec exactly how an image should be decoded.

      • createImageDecoder

        protected com.sun.media.jai.codec.ImageDecoder createImageDecoder(java.io.InputStream src,
                                                                  com.sun.media.jai.codec.ImageDecodeParam param)
        Instantiates a PDSImageDecoder to read from the given InputStream.

        By overriding this method, PDSCodec is able to ensure that a ForwardSeekableStream is used to wrap the source InputStream instead of the a general (and more expensive) subclass of SeekableStream. Since the Vicar decoder does not require the ability to seek backwards in its input, this allows for greater efficiency.
