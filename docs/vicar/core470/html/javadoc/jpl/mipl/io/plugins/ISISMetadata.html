    public class ISISMetadata
extends javax.imageio.metadata.IIOMetadata
implements java.lang.Cloneable
    Version:
    0.5 EA2
    The metdata object is needed by an IIOImage to be a container for the metadata for an image. The reader will create the Document and put it in here. If the "common" IIOMetadataNode verdion of the metedata is requested it will then be built using the native Document as the input.
