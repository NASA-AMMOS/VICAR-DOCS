Class VicarSwapDataInputStream

    java.io.Closeable, java.io.DataInput, java.lang.AutoCloseable

    public class VicarSwapDataInputStream
extends VicarDataInputStream
implements java.io.DataInput
    VicarSwapDataInputStream lets an application read primitive data types from an input stream; the data are converted to Java's integer and real representations (HIGH, IEEE respectively) from LOW/RIEEE int/real representations.
