Class VicarSwapDataOutputStream

  • All Implemented Interfaces:
    java.io.Closeable, java.io.DataOutput, java.io.Flushable, java.lang.AutoCloseable

    public class VicarSwapDataOutputStream
extends VicarDataOutputStream
implements java.io.DataOutput
    VicarSwapDataOutputStream lets an application write primitive data types to an output stream; the data are converted from Java's integer and real representations (HIGH, IEEE respectively) to LOW/RIEEE int/real representations.
