    public class ImageLoader
extends java.lang.Thread
    This utiltity class loads ImageChips from files. It can, optionally, load them into an ImageBag object if one is specified. The loader handles the following types:
    • VICAR image files;
    • JPEG image files;
    • *.lis* files, comprising a plain text list of other files to attempt to load; and
    • directories, where all files in the directory will be loaded.

    By default, a progress bar is displayed to the user for *.lis* and directory loading. This gives the user an idea of the number of files completed and remaining.

    To load a file into a global ImageBag object, you may use code similar to the following: 

         ImageLoader imageLoader=ImageLoader.create("path/to/myFile");
     imageLoader.setImageBag(SVT.getImageBag());
     imageLoader.load();

    NOTE: This code spawns a spearate thread to handle file loading.
