jpl.mipl.mica.util

Class PixelRenderSpeedMonitor

  • java.lang.Object
    • jpl.mipl.mica.util.PixelRenderSpeedMonitor

  • public class PixelRenderSpeedMonitor
extends java.lang.Object
    A static timer used to compute rendering speed. Any ReprojectionOpImage call that causes pixels to be computed can be timed. To use the monitor, start it right before the rendering that you wish to time, then stop it and print the results when the rendering is done. The rendering rate will be computed in pixels per second based on the start and stop times and the number of pixel computations logged.

    NOTE: Pixel computations are done during the painting of a ReprojectionOpImage, not during the JAI setup. To time the JAI overhead as well, start the time before the JAI operation chain has been created.

    An example showing how to use the timer, taken from the MosaicComponent code: 

            PixelRenderSpeedMonitor.startCounting();

        paintComponent(g);

        PixelRenderSpeedMonitor.stopCounting();
        PixelRenderSpeedMonitor.printRate();
