jpl.mipl.mica

Class SVT

  • java.lang.Object
    • jpl.mipl.mica.SVT

  • public class SVT
extends java.lang.Object
    The SVT is the MICA system's central object. It maintains references to the global objects, including the:
    • Editor,
    • Display,
    • ToolPalette, and
    • ImageBag.
    These globals are referenced through the SVT's static methods. When an instance of one of these objects is created, it registers itself as the new global verision of that class of object, replacing any previously registered version.
    Author:
    SGT, Inc.
    See Also:
    Editor, Display, ToolPalette, ImageBag
