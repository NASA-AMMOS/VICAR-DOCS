Help for VIC2SRF

VIC2SRF is a program to convert a VICAR labelled file (or color triplet
of files) to Sun Raster format.  The input may be any of the following:
    (1) a single channel file                   -->  b&w output
    (2) one channel of a multichannel file      -->  b&w output
    (3) three single channel files              -->  color output
    (4) three channels of one multichannel file -->  color output
The output is 8 bit for black-and-white, 24 bit for color.
At the present time BIP organization and all formats other than BYTE
are not supported.

PARAMETERS:

INP

 Input VICAR file name(s) (red, green, blue)

SROUT

 Output Sun raster file name

BANDS

 The band number(s) to use, if the input is a multi- channel image. For color, the order is (red,green,blue).

SIZE

 The size of the outputs if not the entire image.

SL

 The first image line to be output.

SS

 The first sample to be output

NL

 Number of lines output.

NS

 Number of samples output.
