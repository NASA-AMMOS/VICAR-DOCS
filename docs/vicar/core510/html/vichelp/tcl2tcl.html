Help for TCL2TCL

PURPOSE:
tcl2tcl evaluates the passed script in a TCL/Tk shell and returns the
string result, as well as parsed numeric values, if possible. If
returned value is not numeric, realval and string val will be set to
zero.

EXECUTION:

Example:

tcl2tcl script="pwd" strvar=ldir
            where ldir is a TAE TCL variable of type string
          
TIMING: 

Instantaneous.

PARAMETERS:

SCRIPT

 Input TCL/Tk script, up to 99 characters long.

STRVAR

 TAE TCL variable name, string, set to value returned by script evaluation. The result is limited to at most 99 characters.

INTVAR

 TAE TCL variable name, integer, set to integer parse of strvar, or 0 if not.

REALVAR

 TAE TCL variable name, real, set to real parse of strvar, or 0.0 if not. .END
