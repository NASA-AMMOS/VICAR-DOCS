    ThreadGroupNotifier

    public class ThreadGroupNotifier
extends java.lang.ThreadGroup
    A thread group that is used when threads are created. This is a synchronized class that is used to catch exceptions that are not caught by threads in this thread group. It is our recommendation that each thread should be furnished with this class.
