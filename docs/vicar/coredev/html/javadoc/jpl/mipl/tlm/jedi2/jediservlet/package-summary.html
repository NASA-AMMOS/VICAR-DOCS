Package jpl.mipl.tlm.jedi2.jediservlet

  • Class Summary 
    Class Description
    ClientManager
    Class that keeps track of all the clients currently listening for streams.
    ClientObject
    This class contains useful attributes and methods required for each client.
    CloseClient
    Servlet that closes down the client and removes it from the ClientManager.
    CloseStream
    This class closes down the stream, and removes the stream ID from the manager if no clients are listening to it removeStream is called when timer goes off for a specific stream, or when the client is closed.
    GetPersonality
    Servlet which returns the url for the personality file, given a clientID in the requesting url.
    JediControlServlet
    Initializes the JEDI servlet.
    JediInfo
    Servlet that requests user selects if they wish to use the TPS system or read from a directory This should be a replaced with a JSP page
    JediStreamListener
    Telemetry Processing System (TPS) web application
    ManageClientServlet
    Servlet that assists in managing clients
    NextImageServlet
    Servlet that returns the next image to the requesting client.
    RoleJedi
    Checks the roles assigned for a given user id
    SetClientStreams
    Servlet that takes user input and does initialisation of client and other parameters.
    SourceListManager
    Class which sets up file filtering for reading from a directory and obtains file list.
    StandaloneServlet
    Servlet that processes incoming URLs from Cassini standalone telemproc.
    StreamManager  
    StreamObject  
    StreamTimer
    Timer to remove streams after X minutes if no clients are listening Contains inner class which calls CloseStream when timer is triggered.
