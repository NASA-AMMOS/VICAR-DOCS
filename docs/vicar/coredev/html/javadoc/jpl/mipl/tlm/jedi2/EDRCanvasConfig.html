Class EDRCanvasConfig

  • All Implemented Interfaces:
    java.io.Serializable

    public class EDRCanvasConfig
extends CanvasConfig
    A convenient and useful class for storing the salient pieces of a EDRCanvas's configuration. As such, it allows for some flexibility in long-term persistence: this class knows how to write and read a textual representation of itself. This is far better than the previous Serialized object method used previously. Many code changes caused the dreaded serialVersionUID mismatch problem. ASCII shouldn't suffer from this ;-)
