Help for TARGET

PURPOSE:
To create test targets for optical systems from which mtf's can be extracted.

EXECUTION:
target out=(r.img,g.img,b.img) nl=1000 ns=1000

METHOD:

HISTORY:
4-1-99  J Lorre. 
COGNIZANT PROGRAMMER:  Jean Lorre

PARAMETERS:

OUT

 1-3 target images

NL

 Number of output lines.

NS

 Number of output samples.

CYCLE

 Number of wavelengths per circumference. Only used for target # 1.

TARGET

 1 = radial spokes 2 = radial frequencies
