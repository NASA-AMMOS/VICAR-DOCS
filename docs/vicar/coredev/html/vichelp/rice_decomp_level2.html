Level 2 Help for RICE_DECOMP

INP

 
Complete path name of the uncompressed VICAR image to be decompressed.

OUT

Complete path name of the output VICAR decompressed image.

FLAG

Keyword to print details of decompression to standard output.
Tables of decompressed image block packages contents and label
information is provided using this option.
Valid values are VERBOSE, NONE. Default is NONE.
