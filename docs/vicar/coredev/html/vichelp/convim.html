Help for CONVIM

CONVIM is used to read an image from magtape and convert it to VICAR2 file 
format.  It can be used for both labeled and unlabeled images, in byte, 
halfword, fullword, real, or complex format.  All data is converted 
automatically to the proper internal format.

The magtape must be mounted with the TAE MOUNT command before
beginning execution.

PARAMETERS:

NAME

 Symbolic name for tape drive from MOUNT

OUT

 Output file name(s)

FILES

 Tape data set numbers to be read

WINDOW

 Window to read, or size of output

NL

 Number of lines or records in input

RECLEN

 Logical record length of input, in bytes

FORMAT

 Pixel format of each image

LABEL

 (valid: LABEL) Indicates that all output files must have labels
