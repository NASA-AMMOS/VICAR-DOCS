Help for TCL2FILE

PURPOSE
     
CALL
     tcl2file OUT TCLTEXT
  WHERE:
     OUT            is the output data set
     TCLTEXT        are text lines that become text lines in the
                    output file; each is free formatted 
    
OPERATION

PERFORMANCE

Less than 1 second

RESTRICTIONS
------------

Only 100 lines can be input
