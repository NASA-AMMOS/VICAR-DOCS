Level 2 Help for RICE_COMP

INP

 
Complete path name of the uncompressed VICAR image to be compressed.

OUT

Complete path name of the output VICAR compressed image.

SYNCINTRV

Number of compressed blocks stored between synchronization counter
increments. The default value is 1024 stored.

BLOCKSIZE

Number of pixels per block. Valid values range from 4 to 24.
This value must be evenly divisible into NL * NS.  The default 
value is 16 pixels per block.

SEGMTSIZE

Maximum number of compressed bytes per record.  This size is based
on the telemetry downlink size and the reliability of the downlink
stream size.  The default value is 875 bytes per record.

FLAG

Keyword to print details of compression to standard output.
Tables of compressed image block packages contents and label
information is provided using this option.
Valid values are VERBOSE, NONE. Default is NONE.
