Help for DATETIME

PURPOSE: "datetime" types the day and time.  No parameters are needed.

EXECUTION: TAE COMMAND LINE FORMAT     "datetime".

Results in "The date and time are: dd-mmm-yyyy  hh:mm:ss" typed at terminal.
