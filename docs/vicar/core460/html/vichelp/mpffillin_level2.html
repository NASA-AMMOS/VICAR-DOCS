Level 2 Help for MPFFILLIN

INP

 
first: x in meters 
second: y in meters 
third: z in meters

OUT

first: x in meters 
second: y in meters
third: z in meters

SIGMA

The number of standard deviations permitted in each record of range data.
Each picture line of range values is fit to a line relating range to sample.
Points are removed which deviate from the line by F*sigma where F is
provided by the SIGMA parameter; ie: SIGMA=F.
Defaults to 2.5  .

MODE

Controls which algorithm to employ.
1 = Do sigma editing of range values only. 
2 = Do fillin of gores only.
3 = Do both. (the default)
