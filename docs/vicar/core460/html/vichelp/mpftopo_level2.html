Level 2 Help for MPFTOPO

INP

 
First: x image
Second: y image
Third: z image
Fourth: left eye view

OUT

The topography elevation map.

COORDS

Coordinate system for x,y,z points. The choices are:
CAHV = 1 (Camera coordinates centered on the az el pivot point.)
Lander = 2 (Lander coordinates. Z down.)
Mars = 3 (default, Mars Local Level. X north, Z down, Y east)

FORMAT

Output format
BYTE HALF or REAL
Default is REAL

SCALE

Pixels/meter in the output image.
Defaults to 40

CENTER

Four numbers: (X,Y,line,sample)
Where (X,Y) represent the coordinates which map to (line,sample) in the output
picture. Defaults to (0 0 256 256).

DNSCALE

Four numbers: (Z_lo,DN_lo,Z_hi,DN_hi)
Where Z_lo goes to DN_lo in the output and Z_hi goes to DN_hi in the output.
This is a linear stretch.
Defaults to (0,0,1,1).

NLOUT

# lines in output
Defaults to 512.

NSOUT

# samples in output
Defaults to 512.

MODE

Determines the output dn value.
Your choices are either "Z" which places the Z value into the output, or
"DN" which places the DN value (from input 4) into the output.
Defaults to Z.

BAND

The vicar image band number. Defaults to 1
