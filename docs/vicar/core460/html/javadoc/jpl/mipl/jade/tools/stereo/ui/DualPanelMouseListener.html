Class DualPanelMouseListener

  • java.lang.Object
    • java.awt.event.MouseAdapter
      • jpl.mipl.jade.tools.stereo.ui.DualPanelMouseListener
  • All Implemented Interfaces:
    java.awt.event.MouseListener, java.awt.event.MouseMotionListener, java.awt.event.MouseWheelListener, java.beans.PropertyChangeListener, java.util.EventListener

    public class DualPanelMouseListener
extends java.awt.event.MouseAdapter
implements java.awt.event.MouseMotionListener, java.beans.PropertyChangeListener
    Purpose: Implementation of the mouse and mouse motion listeners for JadeDisplay components within a JScrollPane. The constructor accepts the display component and viewport which will trigger mouse events. Updates to coordinates and other mouse state info will then be propagated to the model. Because this listener will be used for left and right views, the constructor also accepts the property with which mouse coodinates will be associated.
