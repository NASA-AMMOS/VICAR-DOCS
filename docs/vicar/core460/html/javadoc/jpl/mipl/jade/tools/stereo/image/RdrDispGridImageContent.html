    public class RdrDispGridImageContent
extends RdrImageContentBase
implements GriddableRdrImageContent
    Purpose: Encapulates the JAI render graph and controls for a disparity RDR file. Includes the bandmerge operator with a constant image at end of render graph, and the associated control panel. Default data range = (0.0, 1024.0)
