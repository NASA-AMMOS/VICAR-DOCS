    public class ContourStretchOpImage
extends PointOpImage
    Purpose: A JAI operator which is used to create a contour stretch from a source image. A ContourStretchOpImage should not be instantiated directly, but through the JAI operation registry.

                     double[] invalidVal,
                     int invalidDn)
        Constructs a ContourStretchOpImage. The image dimensions are copied from the source image. The tile grid layout, SampleModel, and ColorModel may optionally be specified by an ImageLayout object.
        Parameters:
        source - a RenderedImage source
        layout - an ImageLayout optionally containing the tile grid layout, SampleModel, and ColorModel, or null.
        cInterval - interval of minor contours
        cWidth - half-width of interval, from center to edge
        cOffset - offset of contours from origin
        majCFactor - interval of major contours, one major every n minors
        axisDn - axis DN value
        majCDn - major contour DN value
        minCDn - minor contour DN value
        bgDn - background DN value
        invalidVal - value indicative of invalid data
        invalidDn - invalid data DN value
