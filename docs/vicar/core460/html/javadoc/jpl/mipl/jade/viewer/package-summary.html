Package jpl.mipl.jade.viewer Description

Package containing GUI and model aspects of the JADE Image Viewer (JIV) package, including the main program, jpl.mipl.jade.viewer.ImageViewer.
NOTE: The JADE Image Viewer is built upon components from the Mars Image Viewer. In order to use, you must have access to mars.jar.
