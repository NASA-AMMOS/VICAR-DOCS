    Simple program to display an image, and the perspective transform for that image as found by the gruen correlator (e.g. marscor3).

    The transform is: x' = ax + by + c + gxy y' = dx + ey + f + hxy The coef file contains 6 bands, abdegh in that order. The translation terms (c,f) are the disparity and are in the disparity file. Inputs are:

    1) "Left" (reference) image file

    2) "Right" image file. Note that left and right are arbitrary, but the disparity and coefs files map l->r, i.e. they have the "left" geometry.

    3) Disparity file.

    4) Coefficients file. From out_coefs in marscor3.

    5) optional V (Y) size of correlation window, for box cursor

    6) optional H (X) size of correlation window, for box cursor

    As the cursor moves around the left image, the cursor follows in the right image. A box, transformed by coefs, is displayed in the third window.
