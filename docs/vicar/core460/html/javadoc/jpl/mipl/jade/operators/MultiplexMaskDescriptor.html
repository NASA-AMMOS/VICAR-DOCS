    public class MultiplexMaskDescriptor
extends OperationDescriptorImpl
implements java.awt.image.renderable.RenderedImageFactory
    Purpose: A single class that is both an OperatorDescriptor and a RenderedImageFactory for the multiplex data mask operation. An OperationDescriptor describing the "MultiplexMask" operation. The MultiplexMask operation performs a bit shift then bit mask operation on each pixel. Parameters: 1) shiftValue (bit-shift) and 2) maskValue (bit-mask) Operation: pixelOut = (pixelIn >> shiftValue) & maskValue Legal input source images must be of type: TYPE_INT,TYPE_SHORT,TYPE_USHORT,TYPE_BYTE Output image will be of type TYPE_BYTE.
