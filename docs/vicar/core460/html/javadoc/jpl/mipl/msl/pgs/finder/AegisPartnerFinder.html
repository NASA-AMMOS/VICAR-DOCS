    public class AegisPartnerFinder
extends AbstractGroupFinder
    This is the partner finder plugin that attempts to match an AegisTargets data product with the reference image, and vice-versa. The matching is done by comparing BootCounter, CommandNumber, and SequenceId values in the metadata file (emd). This module first registers the input to the database; recording the values for each of the property listed above. The next stage is to query the database for the partner data product that has the BootCounter,CommandNumber, and SequenceId values. The partner returned will always be the highest version that has already been registered. This module only works on the .emd files. When a partner is found, it assumes that a .dat file also exist in the same location as the .emd file. References to both the .dat and .emd files are included in the list of partners returned by the getInputPartners method. This finder plugin requires a database with the AEGIS_INPUTS table and stored procedures. Testing: % java -Djpl.mipl.mars.pipelinedb.host=$PGS_DB_SVR -Djpl.mipl.mars.pipelinedb.db=$PGS_DB_NAME -Djpl.mipl.mars.pipelinedb.port=$PGS_DB_PORT -Djpl.mipl.mars.pipelinedb.dbuser=mslopgs -Djpl.mipl.mars.pipelinedb.dbpass=mslopgs -Djpl.mipl.mars.versionerdb.host=$PGS_DB_SVR_VER -Djpl.mipl.mars.versionerdb.db=$PGS_DB_NAME_VER -Djpl.mipl.mars.versionerdb.port=$PGS_DB_PORT_VER -Djpl.mipl.mars.versionerdb.dbuser=mslopgs -Djpl.mipl.mars.versionerdb.dbpass=mslopgs jpl.mipl.msl.pgs.finder.AegisPartnerFinder $cwd/products/vstb/2014-182/ImgImageLocoNl/ImgImageLocoNl_0457514923-35682-1.emd If partner was previously registered to the database, this execution will return the location of the partner, otherwise "No partners found" is returned. % java -Djpl.mipl.mars.pipelinedb.host=$PGS_DB_SVR -Djpl.mipl.mars.pipelinedb.db=$PGS_DB_NAME -Djpl.mipl.mars.pipelinedb.port=$PGS_DB_PORT -Djpl.mipl.mars.pipelinedb.dbuser=mslopgs -Djpl.mipl.mars.pipelinedb.dbpass=mslopgs -Djpl.mipl.mars.versionerdb.host=$PGS_DB_SVR_VER -Djpl.mipl.mars.versionerdb.db=$PGS_DB_NAME_VER -Djpl.mipl.mars.versionerdb.port=$PGS_DB_PORT_VER -Djpl.mipl.mars.versionerdb.dbuser=mslopgs -Djpl.mipl.mars.versionerdb.dbpass=mslopgs jpl.mipl.msl.pgs.finder.AegisPartnerFinder $cwd/products/vstb/2014-182/AegisTargets/AegisTargets_0457515114-37508-1.emd If executed after the first, this should return the location of the image partner. NOTE: The "DB" environment variables used in this example are set by the "select" in the MSL environment
