jpl.mipl.pgs.utils.logging.server

Interface LogServer_IF

  • All Superinterfaces:
    java.rmi.Remote, ServerBase_IF
    All Known Implementing Classes:
    LogServer

    public interface LogServer_IF
extends ServerBase_IF
    Interface of a RMI server that provides log service. The server is required because multiple applications may be logging to same logger simultaneously so some kind of synchronization is required.
