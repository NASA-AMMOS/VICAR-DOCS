    This package contains base classes for various types of event listeners. The EventScheduler can have one or more EventListeners associated with it. The EventScheduler is responsible for starting and monitoring these event listeners. EventListeners listenes for a particular type of event (such as arrival of file in an input directory). Once an event that satisfies all requirements occurs, it is handed to the EventScheduler which then starts the associated pipeline by handing the event and other parameters to the Controller.
