Class ProcessController

  • java.lang.Object
    • jpl.mipl.pgs.jbpm.exe.ProcessController
  • Direct Known Subclasses:
    ProcessRunner, TaskRunner

    public class ProcessController
extends java.lang.Object
    Provides methods for controlling process execution. This is a convenient class that provides methods for controlling process execution such as pause, stop, etc. Any class that intends to do process controlling must use this class both for correctness and consistency reasons. This class relays heavily on ProcessService class.
