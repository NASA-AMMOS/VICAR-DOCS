Class PGSDbSessionFactory

  • java.lang.Object
    • jpl.mipl.pgs.perstore.hibernate.PGSDbSessionFactory
  • All Implemented Interfaces:
    java.io.Serializable

    public class PGSDbSessionFactory
extends java.lang.Object
implements java.io.Serializable
    creates Session. Obtain a PGSDbSessionFactory with 
     static PGSDbSessionFactory pgsSessionFactory = PGSDBSessionFactory.buildPGSDbSessionFactory();
    and store it somewhere static. It takes quite some time to create a DbSessionFactory, but you only have to do it once. After that, creating DbSession's is really fast.
