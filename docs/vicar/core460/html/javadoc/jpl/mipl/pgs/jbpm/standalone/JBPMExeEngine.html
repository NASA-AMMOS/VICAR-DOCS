  • public class JBPMExeEngine

    Engine that runs the standalone controller. This class uses the same configuration mechanism as PGS/Matis. This class uses DefaultStdController as its default controller. This class uses the log4j configured notification logger named LOG4J_NOTIFICATION_LOGGER_NAME to send any error messages.
