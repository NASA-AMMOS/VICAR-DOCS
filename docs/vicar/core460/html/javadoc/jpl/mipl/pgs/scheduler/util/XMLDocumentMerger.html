Class XMLDocumentMerger

  • java.lang.Object
    • jpl.mipl.pgs.scheduler.util.XMLDocumentMerger

  • public class XMLDocumentMerger
extends java.lang.Object
    Merges two given XML documents or Nodes. This is used by the Command and Configuration framework to merge documents. The merge process depends on the name attribute to correctly identify the elements to be merged. The process of merging adds new elements and attributes to the old document and overwrites modified values in old document with values from new document.

    rrp {rrp@jpl.nasa.gov}
