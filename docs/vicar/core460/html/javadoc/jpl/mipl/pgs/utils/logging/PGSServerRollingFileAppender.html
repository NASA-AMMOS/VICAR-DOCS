    public class PGSServerRollingFileAppender
extends org.apache.log4j.RollingFileAppender
    Appender to be used by ActivatableServerBase. In distributed mode, each server logs to a separate log file, specified using setLogFileName. When all servers are running on same machine, a separate log file is created for Perstore, however, Controller and all SFs messages are logged to the file specified by Controller (almost always Controller.log).
