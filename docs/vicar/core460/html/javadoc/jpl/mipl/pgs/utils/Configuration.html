jpl.mipl.pgs.utils

Class Configuration

  • java.lang.Object
    • jpl.mipl.pgs.utils.Configuration

  • public class Configuration
extends java.lang.Object
    The configuration class used by PGS for parsing and obtaining configuration information. This class groups system properties and SSL properties which can be obtained using getSystemProperties() and getSSLProperties() methods respectively. This is a static class. Multiple calls to configure method(s) with changes to the configuration file or new configuration files will add/replace configuration properties.

      • getPGSConfigDirName

        public static java.lang.String getPGSConfigDirName()
        Returns the path name of the directory containing PGS configuration files. This method first looks for PGS_CONFIG_DIR. If it is not defined, this method looks for environment variable V2JAVA. If V2JAVA is defined, the value of "$V2JAVA/jpl/mipl/pgs/config" (note the missing file separator at the end). In this case, the value of PGS_CONFIG_DIR is set to the value of "$V2JAVA/jpl/mipl/pgs/config". If neither PGS_CONFIG_DIR not V2JAVA is defined, null is returned.
