    public class STDLogger
extends java.util.logging.Logger
implements java.io.Externalizable
    A logger to be used with standalone execution engine. This logger object a specialized type of java.util.logging.Logger. It is made a serializable object because it resides in the jBPM process instances' execution contex, which, when passed from node to node, if it is not a serializable object, the jBPM framework throws exception.
