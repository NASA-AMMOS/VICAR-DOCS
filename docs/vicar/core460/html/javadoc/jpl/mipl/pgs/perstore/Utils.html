      • rs2ht

        public static java.util.Map rs2ht(java.sql.ResultSet r)
                           throws java.sql.SQLException
        This method converts a given result set object to a Map such that each attribute in result set is represented as a key=value in the hashtable where key is the column header as used in table and value is the value of the column.
        Parameters:
        r - The result set that is to be converted to a hashtable object. It is assumed that the result set is non null pointing to the correct row.
        Returns:
        Map array containing the key=value fields of the result set
        Throws:
        java.sql.SQLException - if a database access error occurs

      • rs2htArray

        public static java.util.Map[] rs2htArray(java.sql.ResultSet r)
                                  throws java.sql.SQLException
        This method converts a given result set object to a Map array such that each attribute in result set is represented as a key=value in the hashtable where key is the column header as used in table and value is the value of the column.
        Parameters:
        r - The result set that is to be converted to a hashtable object. It is assumed that the result set is non null pointing to the correct row.
        Returns:
        Map array containing a Map object for each entry in ResultSet r
        Throws:
        java.sql.SQLException - if a database access error occurs
