jpl.mipl.io.codec

Class VicarCodec

  • java.lang.Object
    • com.sun.media.jai.codec.ImageCodec
      • jpl.mipl.io.codec.VicarCodec

  • public final class VicarCodec
extends ImageCodec
    A subclass of JAI ImageCodec that handles the Vicar image.

    Currently the codec defaults to the simplest usual cases, which are:
    1 band is a grayscale image
    3 bands are an RGB image.
    Future enhancements will include vicar specific extentions of com.sun.media.jai.codec.ImageDecodeParam and com.sun.media.jai.codec.ImageEncodeParam
    The encode and decode params will allow the user to tell the codec exactly how an image should be decoded.

      • createImageDecoder

        protected ImageDecoder createImageDecoder(java.io.InputStream src,
                              ImageDecodeParam param)
        Instantiates a VicarImageDecoder to read from the given InputStream.

        By overriding this method, VicarCodec is able to ensure that a ForwardSeekableStream is used to wrap the source InputStream instead of the a general (and more expensive) subclass of SeekableStream. Since the Vicar decoder does not require the ability to seek backwards in its input, this allows for greater efficiency.
