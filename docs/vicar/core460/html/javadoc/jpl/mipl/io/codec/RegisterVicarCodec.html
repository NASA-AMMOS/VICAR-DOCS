    public class RegisterVicarCodec
extends java.lang.Object
implements OperationRegistrySpi
    This class uses the JAI OperationRegistrySpi interface to automatically register the VICAR codec. The updateRegistry() is called during JAI startup in response to an entry naming this class in the "META-INF/services/javax.media.jai.OperationRegistrySpi" file in the jar.
