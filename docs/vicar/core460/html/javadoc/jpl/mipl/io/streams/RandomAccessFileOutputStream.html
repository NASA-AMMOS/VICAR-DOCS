Class RandomAccessFileOutputStream

  • java.lang.Object
    • java.io.OutputStream
      • jpl.mipl.io.streams.RandomAccessFileOutputStream
  • All Implemented Interfaces:
    java.io.Closeable, java.io.Flushable, java.lang.AutoCloseable

    public class RandomAccessFileOutputStream
extends java.io.OutputStream
    This class wraps a RandomAccessFile into an OutputStream. It's unbelievable that this is not in java.io! Nor does com.sun.media.jai.codec have a similar class (unlike the input side, where there is FileSeekableStream).

    No buffering is performed in this class directly, so if someone seeks on the underlying RAF, the next write via this class will go to that location. Likewise, if seek() is called in this class, the underlying RAF will be seek'ed as well, which could affect other users.
