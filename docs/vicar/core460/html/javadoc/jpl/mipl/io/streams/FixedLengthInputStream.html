Class FixedLengthInputStream

    java.io.Closeable, java.lang.AutoCloseable

    public class FixedLengthInputStream
extends java.io.FilterInputStream
    This class wraps another InputStream and gives it a fixed, maximum length. Once the given number of bytes are read, this class will return an End Of File regardless of whether or not the wrapped input stream is at the end of file.

    The maximum length can be changed dynamically. The next read operation will succeed or fail based on the new maximum length.

        Sets the maximum size of this input stream. Subsequent reads will fail if the byte count is larger than this maximum. A maximum of -1 disables the maximum feature, i.e. EOF will be returned only when the real EOF of the input stream is reached.
