Class DataInputStreamWrapper

  • java.lang.Object
    • java.io.InputStream
      • jpl.mipl.io.streams.DataInputStreamWrapper
  • All Implemented Interfaces:
    java.io.Closeable, java.lang.AutoCloseable

    public class DataInputStreamWrapper
extends java.io.InputStream
    This class wraps an object implementing DataInput into an InputStream. Although most objects that implement DataInput are already InputStreams, not all are. Most notably, javax.media.imageio.stream.ImageInputStream from the Image I/O package is this way.

    Only the readUnsignedByte() method is used from the DataInput object.

    It's unbelievable that this is not in java.io! And it's even more unbelievable that InputStream is not an interface, thus avoiding the need for all this garbage.

    Since DataInput provides absolutely NO file positioning, obviously markSupported returns false. Thus this class is recommended as a last resort only.

    See Also:
    RandomAccessFileInputStream
