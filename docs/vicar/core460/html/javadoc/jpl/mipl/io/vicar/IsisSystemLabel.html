Class IsisSystemLabel

  • java.lang.Object
    • jpl.mipl.io.vicar.SystemLabel
      • jpl.mipl.io.vicar.IsisSystemLabel
  • All Implemented Interfaces:
    java.lang.Cloneable

    public class IsisSystemLabel
extends SystemLabel
    This class is a bean that maintains all VICAR system label information. see SystemLabel. This class adds some ISIS specific items which are not a part of a vicar label. The SystemLabel is used by VicarIO to control the reading of iamage data from the file and transfering it to the raster Support for the following ISIS items is added: suffix pixels
    selection of bands, since a cube may hold lots (>3)
    core_base
    core_multiplier
    Not all of the ISIS system label features are implemented yet. special pixels are NOT supported yet
