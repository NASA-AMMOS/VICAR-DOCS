Class RandomAccessFileInputStream

  • java.lang.Object
    • java.io.InputStream
      • jpl.mipl.io.streams.RandomAccessFileInputStream
  • All Implemented Interfaces:
    java.io.Closeable, java.lang.AutoCloseable

    public class RandomAccessFileInputStream
extends java.io.InputStream
    This class wraps a RandomAccessFile into an InputStream. It's unbelievable that this is not in java.io! Note that this is almost exactly what is in com.sun.media.jai.codec.FileSeekableStream, but we have our own in case the codec classes are not available.

      • mark

        public void mark(int readlimit)
        Marks the current position in this input stream. Since we're dealing with a RandomAccessFile, simply remember the position to seek to later.

        Because file.getFilePointer() could possibly throw an exception, we catch it here and set the mark value to -1 (i.e. invalid) so a subsequent reset() would fail. Since mark is not defined to throw exceptions, there's not much else we can do.
