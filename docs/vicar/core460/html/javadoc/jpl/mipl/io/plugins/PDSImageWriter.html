    public class PDSImageWriter

    Version:
    1.0 This version includes transcoder capabilities from vicar image files.
    It might be useful to have transcoders subclass this writer, then it would be able to add the transcoding of new formats and also have access to the writer methods.

        If the image was read in from a vicar file then there is a label which is pushed into the properties of the image.
        One could get that instead of creating a new label. If the image isn't from a vicar file a label MUST be created.
