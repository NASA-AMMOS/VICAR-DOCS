Class LabelEntry

  • java.lang.Object
    • jpl.mipl.io.vicar.LabelEntry

  • public class LabelEntry
extends java.lang.Object
    This class describes a single label entry and how to translate it to/from Bean properties. It is merely a data structure, so all fields are public access.
