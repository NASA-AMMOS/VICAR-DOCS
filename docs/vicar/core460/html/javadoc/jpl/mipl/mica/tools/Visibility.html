    public class Visibility
extends javax.swing.JPanel
implements java.beans.PropertyChangeListener, javax.swing.event.ChangeListener, java.awt.event.ActionListener, java.awt.event.ItemListener
    A Visibility tool is a JavaBean that modifies the alpha transparency of an ImageChip. The tool adds and modifies a JAI operator in an ImageChip's ImageChain. The operator adjusts the chip's alpha-band value, which changes the chip's transparency. The effect is also to change the chip image's ColorModel from Transparency.BITMASK to Transparency.TRANSLUCENT. The tool also includes "flicker" buttons to quickly change the transparency from opaque to invisible.
