    public class Display
extends javax.swing.JPanel
implements java.beans.PropertyChangeListener
    The Display object is the global graphical interface component in the MICA system. It displays and manages imagery mosaics and any overlays. It also handles all property change event handling, including refreshes of the mosaic.

    NOTE: The Display object must work in conjunction with a valid ImageBag object.

      • insertOverlay

        public void insertOverlay(InvisibleOverlay ioverlay)
        Adds an InvisibleOverlay on to the MosaicComponent. The relative order of this overlay with any other previously added overlays is not garaunteed, only that the overlay will be visually in front of the mosaic. If the overlay's shouldActivate() flag is set, then the overlay's mouse listeners are actived and the overlay which previously had the mouse is deactivated.

      • removeOverlay

        public void removeOverlay(InvisibleOverlay ioverlay)
        Removes an InvisibleOverlay from on top of the MosaicComponent. If the overlay had control of the mouse, then it is deactivated and the most recent prior overlay desiring the mouse then will be activated.
