Class RadiometryDescriptor

  • java.lang.Object
    • javax.media.jai.OperationDescriptorImpl
      • jpl.mipl.mica.operators.RadiometryDescriptor
  • All Implemented Interfaces:
    java.awt.image.renderable.RenderedImageFactory, java.io.Serializable, OperationDescriptor, RegistryElementDescriptor

    public class RadiometryDescriptor
extends OperationDescriptorImpl
implements java.awt.image.renderable.RenderedImageFactory
    The Radiometry JAI operator applies radiometric correction to a JAI Rendered chain. A RadiometryModel is passed in via a ParameterBlock.

    This operator wraps the RadiometryModel's correction method so that it may be applied as part of a chain of JAI operations. The RadiometryModel must be properly initialized. The source image is limited to TYPE_USHORT image datatypes.
