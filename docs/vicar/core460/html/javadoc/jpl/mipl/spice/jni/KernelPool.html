Class KernelPool

  • java.lang.Object
    • jpl.mipl.spice.jni.KernelPool

  • public class KernelPool
extends java.lang.Object
    A kernel pool allows the contents of kernels to be searched.

    E kernels should be loaded with the load method, then queried via a Statement obtained from createStatement(), then unloaded via unload or unloadAll

    Author:
    Michael Brady
