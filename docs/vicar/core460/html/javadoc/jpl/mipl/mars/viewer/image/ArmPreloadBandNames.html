jpl.mipl.mars.viewer.image

Class ArmPreloadBandNames

  • java.lang.Object
    • jpl.mipl.mars.viewer.image.ArmPreloadBandNames

  • public class ArmPreloadBandNames
extends java.lang.Object
    Purpose: Class responsible for returning the band names for Arm Preload products. There is a preset list that can be used if namespace inference is enabled. This saves the time of loading the label and examining it. You can force the label to be loaded by setting the system property referred to be Constants.PROP_NAME__ARMPRELOAD_ALWAYS_LABEL to true.
