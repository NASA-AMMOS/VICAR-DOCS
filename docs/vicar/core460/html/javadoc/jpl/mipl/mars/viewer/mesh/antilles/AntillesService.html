jpl.mipl.mars.viewer.mesh.antilles

Interface AntillesService

  • All Known Implementing Classes:
    BaseAntillesService, DebugMeshService, MslOpgsIv2ObjSimpleService

    public interface AntillesService
    Purpose: Interface for the mesher services that will retrieve data and metadata given a WedgeReference. The specifics of the service are abstracted, so one implementation could be a script that converts an original format while another could communicate with a web-service. Prior to making any requests, an observer should be set which will receive notification as metadata and actual data becomes available.
