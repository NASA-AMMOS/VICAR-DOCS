      • getItemNodeValue

        public java.lang.String getItemNodeValue(org.w3c.dom.Element node)
        Returns the value of an Item element node. There are two types of values that may exist within the subtree of the node. The first is if the value is some scalar (single number, string, etc), then the value will be either (1) in the node's 'value' attribute, or (2) the value associated with the node. The other type is a list, which can be represented as a single ordered list,ie '(1,2,3)' or as a sequence of children subitems, where the nth subitem contains the nth element of the corresponding list. In either case, the return string will be a comma-delimited list surrounded by parenthesis.
        Parameters:
        node -
        Returns:
        String value of the node.
