jpl.mipl.mars.viewer.util

Class StretchManager

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.StretchManager

  • public class StretchManager
extends java.lang.Object
    Purpose: This class contains methods for applying various stretches to images with an associated rescale operator. Implemented with use with DeferredData objects in mind, computation of data is postponed until the data is requested by another object.
