Class RdrImageContentBase

  • java.lang.Object
    • jpl.mipl.mars.viewer.image.RdrImageContentBase
  • All Implemented Interfaces:
    java.beans.PropertyChangeListener, java.util.EventListener, RdrImageContent
    Direct Known Subclasses:
    RdrArmReachImageContent, RdrContourStretchImageContent, RdrDispGridImageContent, RdrFormatImageContent, RdrFormatOnlyImageContent, RdrMaskImageContent, RdrReachImageContent, RdrTerrainClassV2ConfidenceImageContent

    public class RdrImageContentBase
extends java.lang.Object
implements RdrImageContent
    Purpose: Encapulates the JAI render graph and controls for a generic RDR file. Generic render graph includes the file load followed by a scale operation. By default, no control panel is available for generic types.
    Properties: Constants.OPTION_ZOOM_FACTOR, Constants.OPTION_ROTATION_ANGLE, Constants.OPTION_FILE_FINDER, Constants.OPTION_STRETCH_TYPE, Constants.OPTION_VIEW_OVERLAYS
