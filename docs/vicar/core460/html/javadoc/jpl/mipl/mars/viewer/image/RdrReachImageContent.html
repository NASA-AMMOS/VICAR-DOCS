  • java.lang.Object
    • jpl.mipl.mars.viewer.image.RdrImageContentBase

    public class RdrReachImageContent
extends RdrImageContentBase
    Purpose: Encapulates the JAI render graph and controls for an IDD reachability RDR file. Includes the ContourStretchOp operator at end of render graph, and the associated control panel.
