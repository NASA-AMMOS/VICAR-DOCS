jpl.mipl.mars.viewer.util

Class FilterUnderlayRetainer

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.FilterUnderlayRetainer

  • public class FilterUnderlayRetainer
extends java.lang.Object
    Purpose: This class works with the RdrResultFilterManager by examining the results of the filters, and checking that any potentially underlying products that may have been removed make it back into the pass set. This is due to a dependency that the underlying image be present among the overlayable products. If an otherwise overlayable product has no underlying image, then overlay is aborted. For example, if a user applies the special filter character as a filtering criterion, then in all likelihood, the black-n-white image products would not pass through. Therefore, the special product would have no overlay behavior. This class queries the result list of products and examines what geometries exist. It then queries the same list for a source type of same geometry, If none is found, then a source is queried from the source set of products. If found, that particular product will be added to the new result set.
