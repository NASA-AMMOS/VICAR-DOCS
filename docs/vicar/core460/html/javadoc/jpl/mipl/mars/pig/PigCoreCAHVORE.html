    public class PigCoreCAHVORE
extends PigCoreCAHVOR
    Implements the CAHVORE camera model core functionality.

    This is an immutable class, in that the vectors can only be set by the constructor and cannot be modified. Create a new instance of the class if you need a modification.

    Author:
    Bob Deen, JPL, Nicholas Toole, JPL Based on Todd Litwin's cmod C++ code
