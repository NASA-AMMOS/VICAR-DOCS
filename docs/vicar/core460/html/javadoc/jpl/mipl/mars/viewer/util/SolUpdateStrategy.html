Class SolUpdateStrategy

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.SolUpdateStrategy

  • public class SolUpdateStrategy
extends java.lang.Object
    Purpose: This strategy encapsulates the logic of updating the application's SOL. Specifically, if the old sol exists within the new sol range, retain it. Otherwise, use the initial value specified by the new sol range.
