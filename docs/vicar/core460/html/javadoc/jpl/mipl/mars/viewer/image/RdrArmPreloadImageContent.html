    public class RdrArmPreloadImageContent
extends RdrFormatImageContent
    Purpose: Encapulates the JAI render graph and controls for a ARM preload RDR file. Includes the bandmerge operator with a constant image at end of render graph, and the associated control panel. Default data range = (0.0, 5.0)
