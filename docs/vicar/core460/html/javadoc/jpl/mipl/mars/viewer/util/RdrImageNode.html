jpl.mipl.mars.viewer.util

Class RdrImageNode

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.RdrImageNode

  • public class RdrImageNode
extends java.lang.Object
    Purpose: Data structure for a set of image contents that are grouped according to some relation. The motivation was stereo imaging, allowing a left and right image to be joined in a single structure. However, the design is open to allow for other relations that might appear in the future. Some constraints do exist for instances of this class. 1) All image contents must be of the same geometry (nominal vs linear) 2) All image contents must be of the same size (full vs thumbnail)
