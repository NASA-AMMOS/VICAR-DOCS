      • setFilter

        public void setFilter(java.lang.String filterName,
             ImageFilter filter)
        Sets or resets filters used by this class to generate list of images for display. Filters are indexed by the filterName parameter, as such, at most one filter can be set. If this method is called with a filterName that is already set, then new filter will replace the exising one.
        Parameters:
        filterName - Name of the filter being assigned
        filter - Instance of ImageFilter to be linked to name, null to remove filter at that name.

        public EdrGroup[] getGroups()
        Returns an array of EdrGroups, each of which defines a set of images that are members of the same group. The set of images is first passed through filters in this instance. Group identification is established by retrieving group ids for each entry using file finder, and is thus defined at the file finder level.
