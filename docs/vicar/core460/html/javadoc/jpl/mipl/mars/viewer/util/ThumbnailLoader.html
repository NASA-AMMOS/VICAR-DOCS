      • ThumbnailLoader

        public ThumbnailLoader(java.util.List edrs,
               ThumbnailLoader subsetLoader)
        Copy constructor when argument has a subset of new loaders edrs. This is a performance convenience, allowing us to retain previous loading work if the new list of edrs is a superset of the older loader.
        Parameters:
        edrs - List of EDRS whose thumbnails should be loaded
        subsetLoader - Instance of ThumbnailLoader whose edrList is a subset of the edrs parameter and has the same file finder.
