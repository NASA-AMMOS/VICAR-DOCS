Interface PropertyManager

  • All Known Implementing Classes:
    ApplicationModel, ArmReachSelectModel, ErrorBandSelectModel, FileFinderSelectionModel, GeometrySelectModel, ImageDataModel, ImageViewModel, InstrumentPlacementBandSelectModel, JmeSceneModel, LogModel, MarsImageViewModel, MissionFilterSelectModel, PropertyManagerImpl, RdrResultFilterManager, RetainResultKeyModel, SolModel, StageSelectModel, StereoState, TerrainClassConfidenceSelectModel, ZoomModel

    public interface PropertyManager
    Purpose: Interface for properties management. This should allow decoupling of components that are interested in application options/properties from the holder of such properties (usually the application model).
