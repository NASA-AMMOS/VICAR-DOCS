Class PigPoint

  • java.lang.Object
    • jpl.mipl.mars.pig.PigVector
      • jpl.mipl.mars.pig.PigPoint
  • All Implemented Interfaces:
    java.lang.Cloneable

    public class PigPoint
extends PigVector
    Equivalent to PigVector. It has a different name to better differentiate between 3-vectors used as a location in space (Point) and ones used as a direction (Vector).

    Because of stupidities in Java, we have to duplicate all the constructors here.
