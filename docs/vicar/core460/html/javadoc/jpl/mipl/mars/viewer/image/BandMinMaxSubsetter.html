Class BandMinMaxSubsetter

  • java.lang.Object
    • jpl.mipl.mars.viewer.image.BandMinMaxSubsetter

  • public class BandMinMaxSubsetter
extends java.lang.Object
    Purpose: Utility class to essentially perform a multi-banded min/max operation. The standard JAI min/max operators perform their respective ops on two images, and never across bands of a single image. Support for a default image is included to handle the case where no bands are selected from the source image.
