jpl.mipl.mars.viewer.finder

Class MarsImageFileFinderFactory

  • java.lang.Object
    • jpl.mipl.mars.viewer.finder.MarsImageFileFinderFactory

  • public class MarsImageFileFinderFactory
extends java.lang.Object
    Purpose: Factory which creates instances of MarsImageFileFinder. Registry is created by loading from file finder info file, a configuration file detailing attributes of existing file finders. The factory will first check to see if the system property file.finder.config is set. If not, then a default configuration file will be used.
