        public boolean isStereoPair(java.lang.String left,
                   java.lang.String right)
        Before, we were calling filefinder.isStereoPair(), but in the cases like Phoenix, that can start to be expensive with repeated calls to getGroupId() for the same files. So, this local implementation uses caching so that a group id is retrieved only once.
