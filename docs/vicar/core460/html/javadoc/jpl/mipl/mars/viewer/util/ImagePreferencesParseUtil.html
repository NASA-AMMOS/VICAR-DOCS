jpl.mipl.mars.viewer.util

Class ImagePreferencesParseUtil

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.ImagePreferencesParseUtil

  • public class ImagePreferencesParseUtil
extends java.lang.Object
    Purpose: Utility class primarily responsible for parsing arguments, inferring type, and adding the constructed value to an image preferences instance. Format of property arguments: TYPE:name=value Where TYPE is the image type, name is the property name, and value is the string representation of a primitive quantity or array. Current, the only supported types are: boolean - true, false int - 12, -2, 0 long - 23L, -3L, 0L double - 1.2, -.3, 0.0 float - 2.3f, -.4f, 0.0f string - default Arrays are formatted as: [val1,val2,...,valN], where the types of valI are all the same.
