Interface LineSampleDisplayConverter

  • All Known Implementing Classes:
    VicarLineSampleDisplayConverter

    public interface LineSampleDisplayConverter
    It seems that Vicar image products use 1 as their first index..., which is a bit contrary to standard image coordinates. As such we need a mechanism to translate between nominal coordinates and specialized display values. This interface represents a simplified set of methods for translation.
