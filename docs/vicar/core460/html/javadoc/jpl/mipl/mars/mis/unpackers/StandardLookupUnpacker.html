Class StandardLookupUnpacker

  • java.lang.Object
    • jpl.mipl.mars.mis.unpackers.AbstractUnpacker

        • jpl.mipl.mars.mis.unpackers.StandardLookupUnpacker

  • public class StandardLookupUnpacker
extends StandardUnpacker
    Extension of the StandardUnpacker which adds the ability to map values to strings. This lookup can be applied to all bands or specific bands. Any unsuccessful lookup results in the String format of the original float value being returned.
