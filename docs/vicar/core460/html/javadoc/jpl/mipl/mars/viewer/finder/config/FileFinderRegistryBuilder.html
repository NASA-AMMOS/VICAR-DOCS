jpl.mipl.mars.viewer.finder.config

Class FileFinderRegistryBuilder

  • java.lang.Object
    • jpl.mipl.mars.viewer.finder.config.FileFinderRegistryBuilder

  • public class FileFinderRegistryBuilder
extends java.lang.Object
    Purpose: Builder for FileFinderRegistry instances. By creating a builder using the static method builder(), client can pass the resulting builder object an XML-file defined according to the Mars Image File Finder Info definition to load specific file finder information for use with the Mars Image Viewer.
