    public class RdrSlopeHeadingImageOverlay
extends java.lang.Object
implements RdrImageOverlay
    Purpose: Encapulates the JAI render graph and controls for overlay with Slope Heading RDR. (range -180 to 180)
    I' = original product H' = clamped slope image (formatted) (1 band)
    S' = constant
