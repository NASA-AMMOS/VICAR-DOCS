Class VicarLineSampleDisplayConverter

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.VicarLineSampleDisplayConverter
  • All Implemented Interfaces:
    LineSampleDisplayConverter

    public class VicarLineSampleDisplayConverter
extends java.lang.Object
implements LineSampleDisplayConverter
    It seems that Vicar image products use 1 as their first index..., which is a bit contrary to standard image coordinates. As such we need a mechanism to translate between nominal coordinates and specialized display values. This implementation handles the addition and subtraction between index spaces.
