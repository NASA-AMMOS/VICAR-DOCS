Class RVF_Append

  • java.lang.Object
    • jpl.mipl.mars.rmc.RVF_Append

  • public class RVF_Append
extends java.lang.Object
    Purpose: Append a set of CS definitions from a generic RMC file to site's master file (e.g. output of correction process). Very similar to RVF_Ingest except it takes an RMC file instead of an EDR list. If the reference is not the enclosing Site, the CS is converted to a Site reference using the most current values and the old reference stored in a derivation element.
