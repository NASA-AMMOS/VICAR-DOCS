    Purpose: Interface defining behavior for mission specification-based information retrieval for products. It captures the functionality that will be shared independent of the chosen file directory structure.

      • getOverlayId

        java.lang.String getOverlayId(java.lang.String imagePath)
        Similar to group id, the overlay id focuses on grouping images that can act as the overlay or underlay image of a particular set. This id should be combined with the group id to ensure uniqueness across a larger superset of images. For example, the most common case will include image geometry.
        Parameters:
        Path - to an image file
        Returns:
        Overlay id of that image as defined by the file finder.
