jpl.mipl.mars.viewer.image

Class RenderedOpLoader

  • java.lang.Object
    • jpl.mipl.mars.viewer.image.RenderedOpLoader

  • public class RenderedOpLoader
extends java.lang.Object
    Purpose: Static class that loads images using a filepath and optional 'use option'. Provides a central point to handle image file loading.
    The loader can use a variety of operators to load the image. One method allows the caller to select the method, whereas another relies on a default method.
    This default can be overridden by specifying a value for the java property "miv.imageloader.use", and assigning a value from the set {codec,plugin}.
