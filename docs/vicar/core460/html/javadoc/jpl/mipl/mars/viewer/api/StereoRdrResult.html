      • isStereoPair

        public boolean isStereoPair(java.lang.String left,
                   java.lang.String right)
        Before, we were calling filefinder.isStereoPair(), but in the cases like Phoenix, that can start to be expensive with repeated calls to getGroupId() for the same files. So, this local implementation uses caching so that a group id is retrieved only once. It assumes that a stereo pair have the same type, instrument, and group id, with the left being a left-eye image, and the same respectively for right.
        Parameters:
        left - Left image
        right - Right image
        Returns:
        True if parameters could be a stereo pair
