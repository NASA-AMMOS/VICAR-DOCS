Class MeshManager

  • java.lang.Object
    • jpl.mipl.mars.viewer.mesh.MeshManager

  • public class MeshManager
extends java.lang.Object
    Purpose: Quite possibly Marsviewer's interface into the whole mesh world. This manager could have access to: 1) RenderSystem including 1a) RenderScene 2) Load manager 2a) lookup of load requests Once a mesh file is selected for display, that request should be passed to this and the MAGIC SHALL HAPPEN.
