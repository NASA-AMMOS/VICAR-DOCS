Class DataRangeModel

  • java.lang.Object
    • jpl.mipl.mars.viewer.ui.DataRangeModel

  • public class DataRangeModel
extends java.lang.Object
    Purpose: Model containing a min and a max DN value. Normally used to set a data range. PropertyChangeEvents are fired when the model changes.
