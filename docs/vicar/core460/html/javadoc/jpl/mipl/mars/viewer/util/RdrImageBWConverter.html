jpl.mipl.mars.viewer.util

Class RdrImageBWConverter

  • java.lang.Object
    • jpl.mipl.mars.viewer.util.RdrImageBWConverter

  • public class RdrImageBWConverter
extends java.lang.Object
    Purpose: Utility class to convert color images to grayscale by extracting the intensity band. If input image is single-banded, it will be returned. If input image has RGB colorspace associated with it, the intensity band will be extracted and returned. Otherwise, the first band will be extracted and returned.
