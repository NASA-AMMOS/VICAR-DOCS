        public static void savePreferences(java.lang.String packageId,
                   java.awt.Frame frame)
        Called when the program ends to try to save window bounds and current directory to user's preferences, so they can be restored the next time the program is run. Since this is not a critcal function, errors are ignored. If an error occurs, the data simply won't be saved.

      • applyPreferences

        public static void applyPreferences(java.lang.String packageId,

        This method is called when the program starts to read and apply the preferences that were saved the last time the program ran. Since this is not a critical funtion, errors are ignored. If an error occurs, preferences will simply not be restored.
