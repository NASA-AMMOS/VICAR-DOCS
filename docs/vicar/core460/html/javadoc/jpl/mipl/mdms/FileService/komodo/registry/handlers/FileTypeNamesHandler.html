    RegistryHandler

    public class FileTypeNamesHandler
extends GeneralErrorHandler
    Implementation of the registry handler to handle file type names retrieved from the file registry. This handler is expected to return list of file type names to the calling objects. This class is derived from the GeneralErrorHandler to take advantage of the automatic error capturing and reporting implementation.
