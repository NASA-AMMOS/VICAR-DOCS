jpl.mipl.mdms.FileService.sigevents

Class SigEvents

  • java.lang.Object
    • jpl.mipl.mdms.FileService.sigevents.SigEvents

  • public class SigEvents
extends java.lang.Object

    Significant events class, used to register significant events to a database, and to log messages to a file and/or standared error. The registration function registerSigEvents is overloaded to take either an object that represents the caller or a string that represents the class name of the caller (if the caller is a static method).
