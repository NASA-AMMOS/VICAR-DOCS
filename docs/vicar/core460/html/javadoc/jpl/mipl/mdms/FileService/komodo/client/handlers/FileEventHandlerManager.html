Class FileEventHandlerManager

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.client.handlers.FileEventHandlerManager
  • All Implemented Interfaces:
    FileEventHandlerRegistryIF

    public class FileEventHandlerManager
extends java.lang.Object
implements FileEventHandlerRegistryIF
    Manages loading of file event handlers and presents a single entry point for clients to acquire those handlers. There are two forms of the constructor. The first checks the value of a system property to determine if handling is enabled. The second offers a parameter to force handlers to be loaded.
