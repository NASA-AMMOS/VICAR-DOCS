        public static int startShutdown(int timeout)
                         throws java.lang.InterruptedException
        Set shutdown flag, and wait for transaction count to go to zero. Return after timeout. If the current thread is interrupted by another thread while it is waiting, then an InterruptedException is thrown. This exception is thrown by the wait method, and if thrown, implies that transactionCountLock has been released.
        Parameters:
        timeout - in seconds. If time less than 1 second, return immediately.
        Returns:
        count of transactions left outstanding after timeout.
        Throws:
        java.lang.InterruptedException - when current thread is interrupted
