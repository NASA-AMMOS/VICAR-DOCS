jpl.mipl.mdms.utils.logging

Interface LoggerPlugin

  • All Known Implementing Classes:
    Log4JPlugin, NullPlugin

    public interface LoggerPlugin
    This interface specifies the required methods for a logging strategy. It is the bridge interface used by the master Logger to bridge application message to the specific logging implementation.
    Version:
