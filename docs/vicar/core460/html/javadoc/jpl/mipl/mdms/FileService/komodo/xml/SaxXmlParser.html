Class SaxXmlParser

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.xml.SaxXmlParser

  • public abstract class SaxXmlParser
extends java.lang.Object
    Abstract base class for SAX XML parser. Currently we are using Apache Xerces-J for our SAX parser, but the code should be portable enough to plug-in other parsers. NOTE: To use this parser class, please pass in the following System parameters: 
        -Dorg.xml.sax.driver=org.apache.xerces.parsers.SAXParser
    -Djavax.xml.parsers.SAXParserFactory=org.apache.xerces.jaxp.SAXParserFactoryImpl
    Version:
    $Id: SaxXmlParser.java,v 1.5 2006/12/13 23:52:53 ntt Exp $
    Author:
    R. Pavlovsky, {Rich.Pavlovsky@jpl.nasa.gov}
