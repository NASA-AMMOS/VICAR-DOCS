jpl.mipl.mdms.FileService.komodo.ui.savannah.subscription.handler

Interface SubscriptionHandler

  • All Known Implementing Classes:
    AbstractSubscriptionHandler, NotificationPanelHandler, SavannahModelUpdateHandler, SubscriptionHandlerProxy

    public interface SubscriptionHandler
    Purpose: Interface for SubscriptionHandlers. Contains handleEvent() for handler behavior and initialize(Element) to initialize state from a DOM Element. This stems from the design of the plug-in architecture that specifies implementations of this interface and required attributes in an XML file.
