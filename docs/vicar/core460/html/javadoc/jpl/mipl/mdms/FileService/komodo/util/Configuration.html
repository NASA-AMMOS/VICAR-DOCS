Class Configuration

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.util.Configuration

  • public class Configuration
extends java.lang.Object
    Reads in a configuration file, provides set/get methods for properties. The following environment variable must be specified on the command line of the java interpreter: -DFEI_ROOT -DFEI_VERSION. If they are not set they are defaulted to the empty string
