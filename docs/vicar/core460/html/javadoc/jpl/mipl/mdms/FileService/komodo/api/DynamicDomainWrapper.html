      • getFileType

        public FileTypeInfo getFileType(java.lang.String groupName,
                       java.lang.String typeName)
                         throws SessionException
        Wrapper function to return filetype metadata. First, the domain object will be checked. If found, the result is returned. Otherwise the filetype is considered managed by this instance. If the associated servergroup has not yet be fully loaded, a dynamic loading of filetype information will be made. If successful, the new instance will be returned. If already loaded, then a matching data object will be returned. If no matching, then a session error is thrown indicating that no type exists.

      • ensureGroupInitializedMagic

        protected void ensureGroupInitializedMagic(java.lang.String groupName,
                               java.lang.String typeName)
                                    throws SessionException
        Magic version of group initialization. If groupName is specified, then this is the same as calling the non-magical ensureGroupInitialized(). If groupName is null and type is non-null, then the default group will be initialized and if that group does not contain the typeName, then all groups will be initialized. If groupName is null and typeName null, then only the default group will be initialized.
        Parameters:
        groupName - Server group name
        typeName - File type name
        Throws:
        SessionException - if session error occurs
