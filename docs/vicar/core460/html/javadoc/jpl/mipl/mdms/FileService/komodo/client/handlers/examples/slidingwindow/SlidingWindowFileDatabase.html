      • getRecordCount

        long getRecordCount(java.lang.String servergroup,
                  java.lang.String filetype)
                    throws java.sql.SQLException
        Returns the number of records in database matching parameters. If both servergroup and filetype are specified, returns count for number of rows in filetype. If only servergroup is returned, returns number of rows in servergroup, across filetypes. If neither parameter is specified, then row count for all rows. If only filetype is specified then 0 is returned.
        Parameters:
        servergroup - Servergroup name, can be null
        filetype - Filetype name, can be null
        Returns:
        Count of rows matching parameters
        Throws:
        java.sql.SQLException
