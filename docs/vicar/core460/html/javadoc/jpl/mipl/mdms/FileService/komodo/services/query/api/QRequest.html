        Constructor for regular expression requests, where dates are used to further discriminate the selected set. Note: An empty Data array implies latest. Otherwise one Date means "since", and two means "between".
        the command modifier is internally set to REGEXP.
        Parameters:
        serverGroup - the server group name
        fileType - the file's type
        directory - the client's current working directory
        command - the command's shortcut (a, g, s, n,....)
        modifier - the command modifier (e.g. regular expression, latest)
        regexp - a regular expression
        dates - the list of date filters
