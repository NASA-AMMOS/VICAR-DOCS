    public class JointAndAcceptStrategy
extends java.lang.Object
implements FileEventHandlerAcceptStrategy
    Purpose: Implementation of FileEventHandlerAcceptStrategy that enables handlers for a set of sub-handlers. The options passed in via initialize is examined for all necessary properties, (servergroup, filetype, output dir) and passes if all of the underlying handlers pass. Default for no handlers is true.
