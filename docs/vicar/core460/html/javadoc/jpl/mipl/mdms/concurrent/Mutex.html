Class Mutex

    Synch

    public class Mutex
extends java.lang.Object
implements Synch
    Implementation of conventional mutual exclusive synch strategy. This class implements the conventional mutual exclusive synchronization strategy. Mutex behaves similar to the Java synchronized block, except that it requires explicit release of the obtained lock. This allows developer to have synchronized block behavior in a method without scope limitation.
