      • ingestResult

        public boolean ingestResult(QResult result,
                   Closeable caller)
        Adds a result to the result queue based on the filetype of the result. The last modification time is also examined, and if the latest, is used to update the internal system query. Note: If the size of all results reaches the maximum allowable value, then this method will block until room has been made for the result.
        Parameters:
        result - Result to be added
        caller - Closeable implementation that, when closed, signals that result ingestion should be aborted

      • ingestResults

        public void ingestResults(java.util.List<QResult> results,
                 Closeable caller)
        Adds a list of results to the result queue based on the filetype of the result. The last modification time is also examined, and if the latest, is used to update the internal system query. Note: If the size of all results reaches the maximum allowable value, then this method will block until room has been made for the result.
        Parameters:
        results - List of Results to be added
        caller - Closeable implementation that, when closed, signals that result ingestion should be aborted
