Class PushSubscriptionClient

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.util.PushSubscriptionClient

    public class PushSubscriptionClient
extends java.lang.Object
implements java.lang.Runnable
    Purpose: Creates and maintains client connection with server, listening for new file events as part of the server's push event mechanism. Listeners can be attached by implementing the SubscriptionEventListener interface.

      • run

        public void run()
        Creates and maintains client subscription session. Initially, a new is constructed. If this attempt fails, operation is aborted. Otherwise, a subscription session is initialized between client and server via requests and results. If an error occurs during the session, the client is killed and a new client session is created.
