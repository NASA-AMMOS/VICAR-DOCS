jpl.mipl.mdms.FileService.komodo.client.handlers

Class FileEventHandlerRegistryLoader

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.client.handlers.FileEventHandlerRegistryLoader

  • public class FileEventHandlerRegistryLoader
extends java.lang.Object
    Loader scans classpath for file event handler plugin descriptors (as an XML file) and attempts to load into handler info instances. Classpath location of individual descriptors is: /META-INF/services/komodo.filehandler.xml, such that no two handlers are packaged within the same Jar file.
