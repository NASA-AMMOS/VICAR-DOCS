jpl.mipl.mdms.FileService.komodo.ui.savannah

Class SavannahDateFilterModel

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.ui.savannah.SavannahDateFilterModel

  • public class SavannahDateFilterModel
extends java.lang.Object
    Purpose: Model class for modeling date filter and controlling state and change notification. Application model is assumed to have a reference to an instance of this.
    Observers of date state must register themselves to this model for updates. Property name is equal to the filter name, as such, care should be taken when selecting a name for a new filter so that it does not conflict with any other application properties.
