    public abstract class FileGetHandler
extends GeneralErrorHandler
    Abstract implementation of the registry handler to handle file get request to the registry. When information is retrieved from the registry, subclasses update the input Result object with the available data. This class is derived from the GeneralErrorHandler to take advantage of the automatic error capturing and reporting implementation.
