      • _extensions

        protected java.util.Map _extensions
        Map from extension point to extension. Plugin extension points are assumed to be unique. Hence, there should never be duplicate extension points defined for a plugin. When there is more than one behavior to be added to a point, then the underlying point element should be a set type containing those.
