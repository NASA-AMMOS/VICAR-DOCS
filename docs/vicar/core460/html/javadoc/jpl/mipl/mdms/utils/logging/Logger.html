jpl.mipl.mdms.utils.logging

Class Logger

  • java.lang.Object
    • jpl.mipl.mdms.utils.logging.Logger

  • public class Logger
extends java.lang.Object
    The common logging service class. This class uses plugin architecture to bridge custom logging strategy. The plugin class can be specified through the property jpl.mipl.mdms.utils.logging.pluginClass . By default this class uses log4j plugin where log format and appenders can be dynamically configured through a properties/xml configuration file.
