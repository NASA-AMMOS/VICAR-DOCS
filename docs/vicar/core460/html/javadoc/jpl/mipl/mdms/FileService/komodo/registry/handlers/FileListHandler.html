    public abstract class FileListHandler
extends GeneralErrorHandler
    Abstract implementation of the registry handler to handle file list retrieved from the registry call. Subclasses of this handler are expected to return a formated message that is compilant with the komodo protocol code for a given API version. This class is derived from the GeneralErrorHandler to take advantage of the automatic error capturing and reporting implementation.
