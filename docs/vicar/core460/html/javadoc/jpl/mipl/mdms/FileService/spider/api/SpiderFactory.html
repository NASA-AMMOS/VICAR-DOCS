        Creates a SPIDER-FEI connector object. This object will guarantee that all SPIDER messages are delivered to FEI and it will collect the results asynchronously. It will also clean up the messages from the localdisk once the are delivered to FEI successfuly.
        Parameters:
        user - FEI username
        passwd - FEI password
        filetypeName - FEI filetype on which the messages will be delivered
        localPath - location on the local disk where the XML messages will be stored
        cleanup - Turns ON/OFF auto-delete of messages from local disk upon successful transfer to FEI.

        SpiderFei object described above
