Class Task

  • java.lang.Object
    • jpl.mipl.mdms.concurrent.Task
  • All Implemented Interfaces:
    java.lang.Runnable

    public abstract class Task
extends java.lang.Object
implements java.lang.Runnable
    This abstract class implements the base class for the Active Object design pattern (POSA2). Objects of this type contain an message queue as communication channel between them when running in multi-threaded environment.
