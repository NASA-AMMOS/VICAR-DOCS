jpl.mipl.mdms.FileService.komodo.ui.savannah.subscription.util

Class PluginLoader

  • java.lang.Object
    • jpl.mipl.mdms.FileService.komodo.ui.savannah.subscription.util.PluginLoader

  • public class PluginLoader
extends java.lang.Object
    Purpose: Loads and initializes plug-ins for the Savannah application. Searches global and local plugin paths, examining directories containing a plugin.xml manifest.
    Search paths include: 
     if system property komodo.global.plugins<\i> set and exists
      value of property ( ${komodo.global.plugins} )
 else
      default global dir ( ${komodo.config.dir}/plugins/ )
 
 if system property komodo.local.plugins<\i> set and exists
      value of property ( ${komodo.local.plugins} )
 else
     default local dir ( ${user.home}/.komodo/plugins/ )
    See PluginInfo for manifest structure.
