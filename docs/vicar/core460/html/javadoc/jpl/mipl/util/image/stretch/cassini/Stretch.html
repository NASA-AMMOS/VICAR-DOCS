jpl.mipl.util.image.stretch.cassini

Class Stretch

  • java.lang.Object
    • jpl.mipl.util.image.stretch.cassini.Stretch

  • public class Stretch
extends java.lang.Object

    Extracted image stretch algorithm from Stretch with significant modifications to make a functional interface to the algorithm using RenderedOp as input and output.

    This class embodies the elements needed to perform contrast modifications to an EDR. There are basically two families of stretches: those which do not require a histogram, and those that do.
