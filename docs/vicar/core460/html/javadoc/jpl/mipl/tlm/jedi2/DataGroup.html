Class DataGroup

    public class DataGroup
extends java.lang.Object
implements java.io.Serializable
    A model bean that that describes an array of data to be displayed in the JEDI client. The current properties used to do this are:
    • a name to associate with this DataGroup
    • a three dimensional array, typically in BSQ organization [band][line][samp]
    • a Histogram. This can be either of the array itself or, more likely, a histogram of the original EDR data
    • the scaling factor used to scale the original DN values to the 8-bit representation used for transfer
    • the organization of the data:
      • BSQ
      • BIL
      • BIP
    • The bands selected from the original data
    Both the client and the server will use this information to perform their function. The client can display a single DataGroup via multiple EDRCanvas views. Changing any of the properties contained int this class will require a new DataGroup for use by the client. I.e., while two EDRCanvas components could show the same data with different stretches, the same cannot be said for two EDRCanvas components displaying a different triplet of bands. This is because the EDRLoader (via the server) will only know to load particular bands for this named DataGroup.
