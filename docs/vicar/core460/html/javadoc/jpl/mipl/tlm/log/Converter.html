jpl.mipl.tlm.log

Class Converter

  • java.lang.Object
    • jpl.mipl.tlm.log.Converter

  • public class Converter
extends java.lang.Object
    Logging converter. This class provides a utility method you can call to convert the MIPL Logging File Structure to an XML document using the logging DTD. It also can be used as a standalone application that takes a single command line argument (the root directory of a MIPL Logging File Structure) and writes (to the standard output) an XML document.

      • convert

        public static void convert(java.io.File file)
                    throws java.io.IOException
        Convert a single log stream directory to XML. This method logs any error messages to the standard error stream, but otherwise silently plods on. It doesn't construct a DOM tree, thereby avoiding any memory problems that can occur with very large raw log files.
        Parameters:
        file - File (/whatever/whatever.log) to convert.
        Throws:
        java.io.IOException
