    public class SFDUTimeoutTask
extends java.util.TimerTask
    Implements the TimerTask Thread for use in TDS Handshake timeouts for SIRTF. This is simply a timer that marks the necessary time before throwing a PropertChangeEvent. It is up to the calling class to determine if a new TimeoutTask is required to fulfill the timeout requirements. In short, this Thread will simply mark the span of time requested.
