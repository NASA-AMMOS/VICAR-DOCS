Class ImageCanvasConfig

  • java.lang.Object
    • jpl.mipl.tlm.jedi.ImageCanvasConfig

  • public class ImageCanvasConfig
extends java.lang.Object
    A convenient and useful class for storing the salient pieces of a ImageCanvas' configuration. As such, it allows for some flexibility in long-term persistence: this class knows how to write and read a textual representation of itself. This is far better than the previous Serialized object method used previously. Many code changes caused the dreaded serialVersionUID mismatch problem. ASCII shouldn't suffer from this ;-)
