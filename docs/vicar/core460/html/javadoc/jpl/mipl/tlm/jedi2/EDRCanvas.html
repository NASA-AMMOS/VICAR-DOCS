    public class EDRCanvas
extends JediCanvas
    EDRCanvas: This class allows for the display of the data in an EDR. To use an object of this class, you'll need to pass the constructor an EDR with an optional Histogram. Additionally, this object will utilize a Stretch object, whereby the image can have constrast modifications performed.
