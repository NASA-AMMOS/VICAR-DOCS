      • setSimulationThreshHold

        public void setSimulationThreshHold(int timeOut)
        Description copied from interface: WorkerHelper
        Sets the simulation thresh hold value. If the timeOut supplied in call to recvSFDU is greater then this value, then the recvSFDU breaks every timeOut seconds indicated by this parameter. This allows checking for isOkayToContiue in some timely manner. Thus solving the problem of ignoring CTR-C when the instrument service is hanging in recvSFDU waiting for more data.
