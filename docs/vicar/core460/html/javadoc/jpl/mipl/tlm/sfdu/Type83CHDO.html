    public class Type83CHDO
extends GenericCHDO
    CHDO for tertiary CHDO, type 83. Typically, this CHDO is used to obtain the SCET of the packet. This CHDO contains the following info (H-channel designations are for SIRTF): 
         |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
   0 |                           chdo_type                           |
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
   2 |                          chdo_length                          |
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
   4 | A | B | C | D | E |         H-0120     packet_apid            |
     |   |   Spare       |                                           |
     |H-0100 - fill flag bit A                                       |
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
   6 |          H-0113    source_pkt_seq_count                       |
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
   8 |          H-0114       non_fill_length                         |
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
  10 |                                                               |
     |                                                               |
  12 |                             spare                             |
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
  14 |                                                               |
     |                                                               |
  16 |     H-0122, H-0123        mm_sclk                             |
     |     coarse   fractions                                        |
  18 |      (note - all packets use same channel unless otherwise)   |
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
  20 |                                                               |
     |                                                               |
  22 |                        SCET                                   |
     |     H-0150 (DAYS) H-0151 (MSECS)                              |
  24 |                                                               |
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|
  26 |      spare                    |  H-0153 scet_flags    | G | H |G = H-0160
     |---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---|H = H-0161
