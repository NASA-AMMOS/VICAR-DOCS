    public class LogArchiveImpl
extends LogArchivePOA
    Log archive. This (executable) class is the MIPL log archive server, accessible through its CORBA interface. Running this class creates an instance of it, optionally registers it with a CORBA naming service, and services requests.

    You may specify the following sysetm properties:

    • jpl.mipl.tlm.log.objectName may contain the name under which this object registers with CORBA naming service. If unset, the server prints its IOR to the standard output. If you specify this property, you'll also need to specify the -ORBnaming ior command-line option or the ooc.service.NameService=ior property.
    • Future properites may be defined, and if so, should be listed here.
