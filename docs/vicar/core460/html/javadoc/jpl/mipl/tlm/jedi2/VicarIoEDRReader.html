Class VicarIoEDRReader

  • java.lang.Object
    • jpl.mipl.tlm.jedi2.VicarIoEDRReader
  • All Implemented Interfaces:
    EDRReader

    public class VicarIoEDRReader
extends java.lang.Object
implements EDRReader
    Creates an EDR object from an EDR on disk using JAI and Bob Deen's/Steve Levoe's VicarIo packages. Basically, to use this object, you simply instantiate it with a VICAR, PDS, or dual-labelled PDS/VICAR file as input, stand back, and hope all goes well.
