MULTI-MISSION INFRASTRUCTURE FACILITY

The Multi-mission Instrument Processing Lab (MIPL) assures a competitive edge by designing/building well-documented and robust ground data systems computing infrastructure environments (hardware and software) that respond to full mission life cycle requirements. The goal of MIPL is to provide a flexible and robust development and operations environment, which leverages integrated and shared resources to support and maintain a project-funded multi-mission infrastructure that reduces instrument software system development, integration and test, and operations costs. This is achieved through shared facilities, common computing resources, display, data storage, data backup, and network resources that provide a common structure for implementation, integration and test (I&T), simulation and/or animation, and data processing and product generation during instrument operations.

The MIPL facility provides an independent software integration and test (I&T) team with recognized expertise and processes in ground system software test program development, automated testing, comprehensive test execution, tracking, and reporting. Additionally, the facility provides a comprehensive software configuration management (CM) capability, structure and methodologies to assure the integrity of software configuration, software builds, and delivery process. The facility also provides expertise in the creation of 2D and 3D visualization hardcopy products, along with associated photo quality large format print capabilities.

Customers taking advantage of the integrated facility consist of software development teams, integration and test (I&T) and configuration management (CM) teams, and mission data processing instrument ops teams.
