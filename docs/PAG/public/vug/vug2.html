2 VICAR Documentation and References

The following list contains various documents which comprise a description of the VICAR system. This includes material describing VICAR from several aspects. Novice and experienced users, as well as programmers, will find their perspectives represented. In addition, information concerning the MIPS facility is provided. In-depth help is available for certain subsets of VICAR (AMOS, BROWSE, etc.) and programs used in conjection with it (dbView, SPICE, etc.).

Key resources for further information are listed below. This document and most of these items are available from:
