1. Introduction

This manual is an introduction to writing software using the VICAR image processing executive and a reference manual for the experienced VICAR programmer. Coding examples are in FORTRAN and C; a knowledge of at least one of these languages is assumed. This manual should be used with “The VICAR User's Guide”: /PAG/public/vug/vugfinal.html and the “VICAR Addendum to the TAE User's Reference Manual”.
