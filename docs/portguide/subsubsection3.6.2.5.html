System Internal Routines

These three routines should never be called by application code. They are called by the ``vicmain_c'' and ``VICMAIN_FOR'' include files to initialize the system. They are listed here because technically they are externally visible routines. However, since only the main include files may ever call them, detailed documentation will not be provided.

  • xvzinit: Do most of the setup for a Fortran main program
  • zvpinit: Initialize parameters from a PARMS parameter file
  • zv_rtl_init: Initialize the Run-Time Library
